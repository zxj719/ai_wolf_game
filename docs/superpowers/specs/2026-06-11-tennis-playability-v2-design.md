# 家庭网球公开赛 · 大型可玩性升级（v2）— 设计文档

日期：2026-06-11
状态：已批准（A→B→C 三段全部实施，每段独立上线）
前置：v1 模块已上线（/tennis，五屏流程 + 全网排行榜，spec 见 2026-06-11-tennis-module-design.md）

## 愿景（用户四条核心诉求）

1. **网球知识融入**：真实网球技术做招式体系，宝可梦式属性克制对战
2. **对战中嵌入小游戏**：操作表现参与胜负，全程"动手"
3. **卡牌 + 地下城**：战术卡牌、随机事件、节点地图
4. **长线养成**：装备系统让低操作玩家也能在长期游玩中获得成就感
5. **双模式**：家族挑战（正统）+ 奇幻闯关（离谱对手、与网球无关的异想天开小游戏，但奖励与结局"圆回网球"）

## 已确认的核心决策

| 决策点 | 结论 |
|---|---|
| 胜负公式 | 混合：招式基础值（属性+装备）× 克制倍率 × 小游戏表现(0.5–1.5) |
| 招式组织 | 宝可梦式：公共池约 8 招，每人固定 4 招位 + 1 专属绝技 |
| 记分系统 | 真实记分 lite：15-30-40（含 Deuce/占先，封顶 2 次）成局，先胜 3 局成盘，三盘两胜 |
| 核心形态 | B 方案：固定招式 + 战术牌库（炉石/StS 式抽牌-手牌-费用）+ 地下城节点地图 |
| 商店 | 卖卡（3 选 1）/ 卡牌升级 / 装备（随机 3 件）/ 装备品质升级 / 移除卡牌 |
| 养成结构 | roguelite 双层：装备/金币/图鉴/成就**永久**，战术卡/加点/体力**单局** |
| 模式 | 家族挑战（6 人梯度连战）+ 奇幻闯关（3 章节点地图 + 离谱对手 + 事件小游戏） |
| 存储 | 登录用户 D1（migration 007 + progress API），游客 localStorage，不做迁移 |

## 一、共享对战内核（A 段）

### 1.1 属性与体力

- 四维属性保留：天赋（反应测试，全局加成 talent×0.4）、力量、技巧、心态（备战加点 + 装备加成）
- **体力条** 0–100 独立资源（基础上限 100，球鞋装备可加）：
  - ≥60 充沛：无惩罚；20–59 疲惫：招式基础值 −20%；<20 力竭：−40% 且耗体 ≥16 的招禁用
  - 局间 +10，盘间 +30；部分招式回体

### 1.2 招式池（8 招）

| # | 招式 | 系别 | 吃属性 | 耗体 | 小游戏 |
|---|---|---|---|---|---|
| 1 | 重炮平击 | 力量 | 力量 | 20 | 节奏连点 |
| 2 | 高压扣杀 | 力量 | 力量 | 16 | 缩圈定格 |
| 3 | 上旋抽击 | 旋转 | 技巧 | 12 | 节奏条 |
| 4 | 切削放缓 | 旋转 | 技巧 | −6（回体） | 精准停条 |
| 5 | 网前截击 | 网前 | 技巧 | 10 | 打地鼠 |
| 6 | 放小球 | 控制 | 心态 | 8 | 力度轻触条 |
| 7 | 挑高球 | 控制 | 心态 | 6 | 方向反应 |
| 8 | 穿越球 | 控制 | 心态 | 14 | 双向时机条 |

### 1.3 克制表（网球知识链，克中 1.5× / 被克 0.7× / 无关 1.0×）

核心四循环：**重炮平击 → 克 → 切削放缓 → 克 → 上旋抽击 → 克 → 放小球 → 克 → 重炮平击**
（力量碾压慢球 / 切削破上旋节奏 / 重上旋让小球来不及 / 小球拉重炮上网）

网前组：**网前截击 克 放小球、切削放缓**；**穿越球、挑高球 克 网前截击**；**高压扣杀 克 挑高球**

每球结算显示克制文案（"效果拔群！切削彻底打乱了上旋节奏"）。

### 1.4 对决流程（每一球）

1. 局首球先做**发球小游戏**（时机条：完美=ACE 直接得分，好=本球 +15% 威力，失误=二发 −10%）
2. **抽牌与打牌阶段**：抽 1 张入手，战术点 +1（上限 3），可打任意张付得起的手牌
3. 双方暗选招式（对手按性格权重 + 体力约束出招，出招前有 75% 真实率的"小动作提示"）
4. 玩家招式触发对应**小游戏**（2–4 秒），表现映射 0.5–1.5 倍率；对手小游戏表现用 d20 折算
5. 结算：`威力 = (吃的属性 + 装备加成 + talent×0.4) × 体力档惩罚 × 克制倍率 × 小游戏倍率 + 卡牌修正 + 微噪声(±3)`，高者得分
6. 真实记分推进：15→30→40→局（Deuce 平分/占先，封顶 2 次后金球制）；3 局成盘；三盘两胜

### 1.5 战术牌库系统（炉石/杀戮尖塔式）

- **牌库**：单局/单次闯关内构建。家族挑战初始牌库 6 张基础卡；闯关初始 5 张，沿路获卡扩充；单局快打固定发 4 张体验牌库
- **抽牌**：每球开始抽 1 张，手牌上限 5（爆牌弃置）；牌库抽空后弃牌堆洗回成新牌库（StS 规则）
- **战术点**：独立费用资源——开局 1 点，每球 +1，上限 3；出招前可打任意张付得起的手牌。与体力解耦（卡牌不和强力招式抢资源）
- **卡牌升级**：每张卡有「+」强化版，商店付费升级

| 费用 | 卡牌（基础 → 升级+） |
|---|---|
| 0 | 深呼吸（回 5 体 → 回 10）；观众助威（下个小游戏窗口 +30% → +50%） |
| 1 | 毛巾时间（+30 体 → +50）；换新球（下招威力 +20% → +35%）；教练手势（透视对手下一招 → 透视+本球窗口 +15%） |
| 2 | 鹰眼挑战（本球输后翻判重打，一局 1 次 → 一盘 2 次）；拍线调整（本球克制无效 → 克制反转）；心理按摩（免疫下次被克 → 免疫 2 次） |
| 3 | 能量胶（耗体减半 3 球 → 5 球）；金球时刻（本球小游戏倍率下限提到 1.0 → 1.2） |

来源：闯关事件/对战奖励三选一（StS 式）、商店、家族挑战每场开局发 1 张。

## 二、永久养成层（B 段）

### 2.1 装备（5 槽 × 4 品质：普通/精良/史诗/传说）

- 球拍 → +力量（2/4/7/12）；球鞋 → +体力上限（6/12/20/30）；手胶 → +技巧（2/4/7/12）；护腕 → +心态（2/4/7/12）
- 挂件 → 特效词条（开局多抽 1 战术卡 / 克制倍率 +0.1 / 小游戏窗口 +15% / 局间回体 +5 / ACE 威力 +25%）
- 掉落：每场对战结束必掉 1（胜=高一档概率，败=保底普通~精良）；事件节点掉落；商店购买
- **装备升级**：商店可将任一槽位当前装备品质 +1（普通→精良 50 金币 / →史诗 120 / →传说 300），手残玩家的金币长线投资通道
- 同槽位自动保留更高品质（无背包管理，降低复杂度）；金币来源：对战胜利/事件/出售重复装备

### 2.2 成就与图鉴

首胜 / 家族球王 / 绝技全图鉴(7) / S 级天赋 / 零失盘夺冠 / 闯关通关 / 传说装备首次入手 / ACE 大师(累计 10 记)。

### 2.3 数据与 API

- `migrations/007_tennis_v2.sql`：表 `tennis_progress`（user_id PK、coins、equipment JSON、unlocked_moves JSON、achievements JSON、championships、adventure_clears、updated_at）
- `GET /api/tennis/progress`（auth）/ `PUT /api/tennis/progress`（auth，服务端白名单校验：装备/成就/绝技 id 枚举内、coins 增量 ≤ 单场上限、championships 单调递增）
- 全网排行榜加「👑 球王次数 / 🗺️ 闯关通关」列；游客全部 localStorage（key `tennis_v2_progress`）

## 三、模式一：家族挑战（B 段）

- 入口：选角后选模式（单局快打 / 家族挑战 / 奇幻闯关）。反应测试与备战每次挑战只做一次。
- 其余 6 位家人按固定难度带连战：第 1 场属性 40–50 → 每场 +8 递增 → 第 6 场 80–90。
- 赛间：体力回至 min(上限, 当前+40)，三选一：特训（随机属性 +8）/ 按摩（额外 +30 体力）/ **进店**（购卡/升卡/购装/升装各一次机会）；每场开局发 1 张随机战术卡。
- 输一场即止步（结算已得装备/金币/解锁）。击败谁解锁谁的绝技（图鉴）；6 连胜 = 家族球王 👑（championships+1）。

## 四、模式二：奇幻闯关（C 段）

### 4.1 世界观与"圆回来"机制

家族奖杯被「网球之神」偷走，穿越三章夺回：**第一章 菜市场江湖** → **第二章 修仙界** → **第三章 太空站**，终点 BOSS 战。**离谱小游戏的奖励全部映射网球能力**（+属性/装备/金币/战术卡），最终 BOSS 是正经网球对决——一路所得全部用上，剧情圆回"家庭网球公开赛"。

### 4.2 节点地图

每章 4–5 个节点、2–3 条分叉路（生成时随机），节点类型：⚔️ 对战 / ❓ 事件（小游戏或剧情二选一）/ 🛒 商店（购卡 3 选 1、卡牌升级、装备 3 件、装备品质升级、移除卡牌——StS 式五项服务）/ 🏕️ 休息（回体力 50 或移除一次疲劳）。每章末固定精英对战，第三章末 BOSS。

### 4.3 离谱对手（复用对战内核 + 规则扭曲）

| 对手 | 章节 | 扭曲规则 |
|---|---|---|
| 广场舞大妈 | 1 | 心理战系招式对她无效（心态 99）；蒲扇旋风=改名版重炮 |
| 外卖小哥 | 1·精英 | 你的小游戏时限 −30%（他赶时间，你也得快） |
| 太极宗师 | 2 | 你的力量系招式 30% 概率被"借力打力"反弹 |
| 修仙童子 | 2·精英 | 每 3 球"渡劫"：双方强制比拼心态 |
| BOT-3000 | 3·精英 | 无心态条不吃心理招，但出招 100% 规律（读招盛宴） |
| 网球之神 | 3·BOSS | 属性 90+，每盘更换一套配招，持双绝技 |

### 4.4 事件小游戏（与网球无关但奖励圆回来）

火锅捞丸子（限时反应点击 → +技巧）/ 躲蒲扇（节奏躲避 → +心态）/ 失重颠球（平衡微调 → +体力上限 5，本次闯关有效）/ 打坐数息（精准计时 → 金币）。每个 10–20 秒，纯前端实现。

### 4.5 闯关的 roguelite 规则

属性加点、战术卡、临时体力上限**单次闯关有效**；装备、金币、图鉴**永久结算**（失败也保留）。通关 adventure_clears+1。

## 五、技术架构

```
src/modules/tennis/
├── battle/            # A 段：共享对战内核
│   ├── moves.js           # 招式池/克制表/绝技（纯数据）
│   ├── cards.js           # 战术牌定义/升级版/掉落池（纯数据）
│   ├── battleReducer.js   # 每球状态机（含抽牌/战术点/打牌，随机数注入纯函数）
│   ├── scoring.js         # 真实记分 lite 状态机（纯函数）
│   └── minigames/         # 小游戏组件（每个输出 0.5–1.5 倍率）
├── meta/              # B 段：永久层
│   ├── equipment.js       # 装备定义/掉落表（纯数据）
│   └── progressStore.js   # D1/localStorage 双写适配
├── modes/             # 运行层
│   ├── ladderReducer.js   # 家族挑战推进（B 段）
│   └── adventure/         # C 段：地图生成/事件/离谱对手数据
└── （v1 文件保留：单局快打沿用新对战内核）
```

- 三层 reducer 隔离：battleReducer（球）/ scoring（局盘）/ ladder·adventureReducer（赛程），全部随机数注入纯函数，沿用 v1 测试模式
- 小游戏统一契约：`<Minigame onDone={(multiplier: 0.5–1.5) => void} timeScale={1} windowBonus={0} />`，离谱对手的规则扭曲通过 props 传入
- 后端仅 B 段新增 progress 两端点 + migration 007；A/C 段零后端改动

## 六、错误处理与边界

- progress 上传失败：本地仍写入，下次进入模块重试同步（last-write-wins，服务端校验白名单防脏数据）
- 中途退出闯关/挑战：进度存 sessionStorage 快照，重进恢复或放弃
- 小游戏在低性能设备掉帧：所有判定窗口基于时间戳而非帧数；`prefers-reduced-motion` 用户自动获得窗口 +30% 补偿
- 游客模式全功能可玩（含闯关），仅榜单与云同步缺失

## 七、测试与验收

- 纯函数测试：克制表全组合、记分状态机（Deuce/金球/盘点）、体力分档、绝技/战术卡效果、梯度生成、地图生成合法性（必有路径到 BOSS）
- 小游戏：时间戳判定的单元测试（模拟 performance.now）
- E2E（puppeteer-core）：A 段单局完整一场；B 段家族挑战打 2 场 + 装备掉落断言；C 段闯关走 1 章
- 每段上线走 CLAUDE.md 部署校验（fingerprint + localhost 泄漏 0）
