## 优化目标

### 猎人 DAY_SPEECH 两连+三连平安夜推断

**背景**：R131 实现了猎人单夜平安夜推断（枪靶排除逻辑），但建议「下轮加两连版本」。  
其他角色（守卫/狼人/预言家/女巫/村民/骑士/摄梦人/魔术师）均已覆盖三层平安夜推断；  
猎人仅有单夜版本，是平安夜推断矩阵中的最后一个缺口。

**博弈论依据**（Wang 2025 arxiv:2408.17177）：

渐进证据积累原则：
- 单夜平安夜高票存活 → 可能被守护（confidence 下调 15-25）
- 两夜高票存活重合 → 极可能被连守（confidence 下调 30-40，枪靶候选移除）
- 三夜高票存活两次以上重合 → 几乎确认好人（confidence 下调 35-45，枪靶彻底排除）

猎人枪靶分析的核心约束：被狼重点针对且被守/救 = 更可能是好人，不是合适枪靶。  
多轮平安夜的重合证据积累大幅降低误射风险，与其他角色（守护目标确认/威胁降低）推断方向相反但结构对称。

**实现细节**（Prepend Injection 第 26 次）：

```javascript
// 新增三个外层变量
const isPeacefulNightHunter = ctx.dayCount > 1 && ctx.lastNightInfo?.includes('平安夜');
const isConsecutivePeacefulHunter = ctx.dayCount >= 3 && isPeacefulNightHunter && ctx.fullGameTimeline?.includes(`N${ctx.dayCount - 2}:平安夜`);
const isTripleConsecutivePeacefulHunter = ctx.dayCount >= 4 && isConsecutivePeacefulHunter && ctx.fullGameTimeline?.includes(`N${ctx.dayCount - 3}:平安夜`);

// if 块内：三连 → 前置到两连头部 → 前置到单夜 step 头部
let tripleConsecutivePeaceHintHunter = '';
if (isTripleConsecutivePeacefulHunter) {
    tripleConsecutivePeaceHintHunter = `⭕【猎人三连平安夜三阶推断...
- 三轮高票存活者若两次以上重合 → 三阶确认：该玩家几乎可确定为好人，立即从枪靶候选彻底清除
  → confidence 下调 35-45，reason 追加"三连平安夜：三连高票存活=极高概率被连守，枪靶彻底排除"`;
}
const consecutivePeaceHintHunter = isConsecutivePeacefulHunter
    ? `${tripleConsecutivePeaceHintHunter}⭕【猎人两连平安夜二阶推断...
  → confidence 下调 30-40，reason 追加"两连平安夜：两连高票存活=极可能被连守，枪靶候选移除"`
    : '';
hunterPeaceNightStep = `${consecutivePeaceHintHunter}⭕【猎人平安夜推断（单夜）...`;
```
