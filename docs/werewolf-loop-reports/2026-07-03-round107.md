```javascript
// R107: NIGHT_DREAMWEAVER 平安夜双来源推断
const isNightPeacefulDW = ctx.dayCount > 1 && ctx.lastNightInfo?.includes('平安夜');
const dwNightPrevDay = ctx.dayCount > 1 ? ctx.dayCount - 1 : 0;
const isConsecutivePeacefulNightDW = ctx.dayCount >= 3 && isNightPeacefulDW &&
    ctx.fullGameTimeline?.includes(`N${ctx.dayCount - 2}:平安夜`);
const dwNightPrevPrevDay = ctx.dayCount >= 3 ? ctx.dayCount - 2 : 0;
const isTripleConsecutivePeacefulNightDW = ctx.dayCount >= 4 && isConsecutivePeacefulNightDW &&
    ctx.fullGameTimeline?.includes(`N${ctx.dayCount - 3}:平安夜`);
const dwNightThreePrevDay = ctx.dayCount >= 4 ? ctx.dayCount - 3 : 0;
let tripleConsecutivePeaceNightHintDW = '';
let consecutivePeaceNightHintDW = '';
if (isConsecutivePeacefulNightDW) {
    tripleConsecutivePeaceNightHintDW = isTripleConsecutivePeacefulNightDW ? `...三阶...` : '';
    consecutivePeaceNightHintDW = `${tripleConsecutivePeaceNightHintDW}⭕【两连平安夜二阶双来源推断...】`;
}
const dwNightPeaceStep = isNightPeacefulDW
    ? `${consecutivePeaceNightHintDW}⭕【摄梦人平安夜双来源推断...】` : '';
```

### Return 模板注入位置
```
${dreamweaverNightStyle}
${dwNightPeaceStep}Step1:（分析目标选择）
```

### 摄梦人双来源推断的独特性
其他角色（预言家/守卫/女巫等）的平安夜推断只有一个来源：守卫的守护目标。  
摄梦人有两个独立来源：
- **来源 A**：狼人攻击了摄梦人的入梦目标 → 被入梦免疫拦截（信息：入梦目标可能是狼刀目标）
- **来源 B**：狼人攻击了其他目标 → 守卫/女巫保护生效（信息：摄梦人入梦目标不是狼刀目标）

三阶版本（三连平安夜）需要对三种路径做推断：
- **路径 A**（全三夜锁守）：守卫连续三夜守同一人，可信度最高
- **路径 B**（高频连守）：前两夜 A 来源，第三夜 B 来源（或相反）
- **路径 C**（随机轮守）：三夜不同来源组合，推断守卫轮守策略
