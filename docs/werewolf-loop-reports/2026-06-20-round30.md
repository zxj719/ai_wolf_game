## 2. 本轮优化详情

### 问题发现

LEARNINGS.md item 58（R29 下轮建议第2项）：猎人上警后竞选发言走"好人 fallback"模板，既无法展示猎人独有的战略底气，也可能因为提示词里暗示具体技能而直接暴露身份。

原始 fallback 模板（猎人走此路径）：
```
你是好人（猎人）：竞选发言 = 分析能力展示，说清3点让警下信服你：
1. 昨夜局势分析 ...
2. 票型判断 ...
3. 警徽流计划/价值主张
```

**问题**：完全没有利用猎人最独特的竞选优势——"死亡不是纯损失"这一隐性威慑。好人 fallback 和真正好人完全一样，猎人的额外博弈价值被丢弃。

### 博弈论依据

猎人在竞选时有独特优势（Wang 2025 信息博弈视角）：
- **死亡不对称**：猎人被刀/被投出 = 开枪反杀一人。狼人刀警长猎人的预期代价远高于刀普通好人。
- **不确定性价值**：若猎人在竞选发言中暗示"我倒了好人不亏"但不说明机制，狼人面临"他是猎人还是有底气的好人"的不确定性，这个不确定性本身就是战略资产。
- **隐性比显性更优**：一旦猎人明说身份，不确定性消失，狼人可精确计算刀猎人的代价；保持模糊则让对方陷入更大的风险估算误差。

### 修复方案

在 `SHERIFF_SPEECH` case 的 `ssHint` 三元链中，在 `playerRole === '狼人'` 之后、好人 fallback 之前插入猎人专属分支：

```javascript
: playerRole === '猎人'
? `你是猎人：竞选发言 = 展示"持续威慑力"——让警下相信投你不亏，但**绝不能暴露猎人身份**。
【隐性威慑框架（改编成自己的语言，不要照抄）】：
① "不惧风险"角度："我认为好的警长就应该是那种不怕被针对的人——即便最坏的情况发生，好人阵营也不会因此白亏"
② "勇于承担"角度："我愿意成为场上风险最高的那个人——警长应该站在最前面，而且就算倒下了，大家也不会白亏"
③ 关键原则：措辞越模糊，威慑越大——狼人无法确认你是猎人还是特别有底气的好人，这个不确定性本身就是策略价值。绝不能明说身份或技能细节，一旦直接点破反而丧失隐性威慑。
【竞选3要点（必须包含实质分析，不要全用威慑角度替代）】：
1. 昨夜局势分析：你认为最可疑的是谁、最信任谁（同其他候选人，展示分析能力）
2. 隐性威慑角度：从①②中选一个，改编成符合当前局势的表达（不超过1句话）
${badgeFlowLine}
结尾可带一句底气话——让自信本身说服别人，而非依赖明确的身份主张。`
```

### T10 修复记录：负向示例词汇激活问题

初版提示词含 `绝不能说"我是猎人"或"我死了能开枪"`——"开枪"出现在负向语境中，但根据 LEARNINGS.md R1（负向词汇列表激活 LLM 注意力），即使写"不能说X"，LLM 仍对 X 保持高激活态。T10 测试（检查猎人分支无"开枪"字样）立即捕获到此问题。

修复：将具体禁止词替换为抽象描述：`绝不能明说身份或技能细节，一旦直接点破反而丧失隐性威慑。`
