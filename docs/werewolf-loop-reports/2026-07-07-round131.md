### B. 猎人 DAY_SPEECH 平安夜推断步骤（Priority C — 补全 DAY_SPEECH 角色覆盖）

**问题识别**：

R130 建议「考察 DAY_SPEECH 专属分支现状」。逐角色检查后发现：
- 狼人/预言家/女巫/守卫/村民：均有 1/2/3 连平安夜推断步骤（R80-R105 系列）
- 骑士/摄梦人/魔术师：在 rolePrompts 模块中（独立维护）
- **猎人（hunter）：独缺平安夜推断步骤** ← 本轮修复目标

**博弈论依据**（Wang 2025 arxiv:2408.17177）：

猎人的平安夜推断视角与其他角色完全不同：
- 其他角色：推断"谁被保护了"→ 更新嫌疑度
- 猎人独有：推断"我的枪靶候选被保护了吗" → 枪靶排除反证

具体逻辑：
```
平安夜高票存活者 = 昨夜被狼刀却被守/救的玩家
若该玩家在猎人的「开枪优先级：高」候选中 
→ 被狼重点针对 = 更可能是好人，不是合适枪靶
→ confidence 下调 15-25，reason 追加降级标记
```

**实现细节**：

在 `hunterSpeechLen` 计算之后新增：
```javascript
// R131: 猎人平安夜推断（枪靶排除逻辑）
const isPeacefulNightHunter = ctx.dayCount > 1 && ctx.lastNightInfo?.includes('平安夜');
let hunterPeaceNightStep = '';
if (isPeacefulNightHunter) {
    const prevDay = ctx.dayCount - 1;
    hunterPeaceNightStep = `⭕【猎人平安夜推断（thought 中完成；speech 只说"平安夜，继续分析局势"即可）】
- 查 D${prevDay} 投票记录中票压最高的存活玩家——最可能是昨夜被狼刀但被守/救的目标
- 枪靶排除推断：若你的开枪候选（identity_table "开枪优先级：高"玩家）与该高票存活者重合
  → 被狼重点针对 = 更可能是好人，不是合适的枪靶
  → identity_table 该玩家 confidence 下调 15-25，reason 追加"D${prevDay}平安夜：高票存活=被保护好人推断，枪靶优先级降级"
- 若开枪候选不在高票存活者中：维持现有 identity_table 判断，平安夜对枪靶分析无新增信息\n`;
}
```

模板插值：`${hunterDayHistoryStep}\n${hunterPeaceNightStep}Step1:...`

同时修正原有发言策略2的措辞：
- 旧: `只说"平安夜，信息有限"就够了`
- 新: `只说"平安夜，继续分析局势"就够了`（与平安夜步骤 speech 指导一致）
