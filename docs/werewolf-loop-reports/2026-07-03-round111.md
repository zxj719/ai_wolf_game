## 实现细节

### 选定优化项
**LAST_WORDS 遗言背景上下文**（R110 报告的首选下轮优先项之一）

研究依据：Wang 2025 (arxiv:2408.17177) - 结构化历史上下文显著提升 AI 决策精准度。遗言是局中信息传递的关键节点，缺乏上下文的遗言质量直接影响好人阵营的追查能力。

### 代码位置
`src/services/aiPrompts.js`：`LAST_WORDS` case block

### 新增变量（共 7 个外层 + 1 个 if 块内）

```javascript
// R111: 遗言背景上下文 - 当前天次 + 先行出局玩家 + 被投出时的票型摘要
const lwDay = gameState?.dayCount ?? 1;
const priorDeaths = (gameState?.deathHistory || [])
    .filter(d => d.playerId !== currentPlayer?.id);
const lwDeathsText = priorDeaths.length > 0
    ? priorDeaths.map(d => `${d.playerId}号(D${d.day}${d.phase ?? ''})`).join('、')
    : '（无先行出局玩家）';
const isVotedOut = (cause || '').includes('投');
let lwVotedByText = '';
if (isVotedOut) {
    const lastVoteRound = (gameState?.voteHistory || []).slice(-1)[0];
    const votesAgainstMe = (lastVoteRound?.votes || [])
        .filter(v => v.to === currentPlayer?.id);
    if (votesAgainstMe.length > 0) {
        lwVotedByText = ` | 投你出局：${votesAgainstMe.map(v => `${v.from}号`).join('、')}`;
    }
}
const lwContextBlock = `【D${lwDay}出局背景】${cause ?? '死亡'}${lwVotedByText} | 先行出局：${lwDeathsText}`;
```

### Return 模板修改

```diff
- return `你已死亡（${cause}），现在是你的遗言时间——这是你最后一次发言，之后不再参与游戏。
+ return `你已死亡（D${lwDay}，${cause}），现在是你的遗言时间——这是你最后一次发言，之后不再参与游戏。
+${lwContextBlock}
 ${lwRoleHint}
 要求：80字以内，信息密度优先，不要煽情告别。
 输出JSON:{"speech":"遗言内容","thought":"你的真实想法"}`;
```

### 关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 先行出局格式 | `X号(DY夜/投票)` | 包含天次和死亡类型，AI 可推断信息演进顺序 |
| 空状态回退 | `（无先行出局玩家）` | 首位出局玩家无先行死亡，需明确告知 |
| 被投出检测 | `cause.includes('投')` | 匹配'被公投出局'，简单可靠 |
| voteHistory 键 | `v.to` （非 v.targetId） | R96 教训：recordVoteRound 已映射为 from/to 格式 |
| 遗言字数 | 维持 80字 | 不扩展，避免影响游戏节奏 |
| 白熊效应 | 零负向禁词 | 全正向描述，T17 验证 ✅（第 32 次验证） |

## 遗言背景上下文实际效果（场景举例）

### 场景 A：D3 投票出局（5号被3、7、8号投出）

优化前：`你已死亡（被公投出局），现在是你的遗言时间……`

优化后：
```
你已死亡（D3，被公投出局），现在是你的遗言时间——这是你最后一次发言，之后不再参与游戏。
【D3出局背景】被公投出局 | 投你出局：3号、7号、8号 | 先行出局：1号(D1夜)、4号(D2投票)
（后接角色专属 lwRoleHint）
```

AI 现在可以：
- 知道自己在第 3 天死
- 知道 3、7、8 号投了自己（潜在对立面 / 狼队嫌疑）
- 知道 1 号 D1 夜死（被狼刀），4 号 D2 被投出
- 基于以上信息给出有针对性的遗言

### 场景 B：首夜 D1 白天公布（1号被刀）

优化后：
```
你已死亡（D1，被狼人杀害），现在是你的遗言时间……
【D1出局背景】被狼人杀害 | 先行出局：（无先行出局玩家）
```

AI 知道自己是首个出局，无历史参考，需完全依赖夜间观察。
