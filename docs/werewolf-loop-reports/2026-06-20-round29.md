## 2. 本轮优化详情

### 问题发现

`SHERIFF_RUN` case 的角色策略提示存在严重合并问题：

```javascript
// 旧代码：猎人与女巫/守卫被一刀切归为"不上警"
: (playerRole === '女巫' || playerRole === '猎人' || playerRole === '守卫')
? `你是${playerRole}：标准打法是不上警隐藏身份（神职暴露会被狼优先刀），除非局势需要你站出来。`
```

此提示词的问题：
1. **猎人被错误归类**：猎人被刀必然开枪，获得警徽后死亡 = 1.5票权重 + 开枪 + 警徽移交，是三重威慑。"标准不上警"与猎人的博弈价值背道而驰。
2. **女巫无例外条件**："除非局势需要你站出来"过于宽泛，AI 无法从中推断出具体的例外判断标准。
3. **守卫与女巫等权**：守卫的隐藏价值（"狼人不知道守护了谁的不确定性"）比女巫更纯粹，应有更强的"不上警"建议。
4. **无博弈依据**：仅说"暴露会被狼优先刀"，没有解释为什么这种成本对猎人而言并不适用（猎人无论如何都会被刀，早晚的区别而已）。

### 修复方案（let + if 块，per R15 教训）

将单个条件表达式拆分为 5 个独立的 `if/else if/else` 分支：

#### 猎人（重点调整）
```
你是猎人：上警是值得认真考虑的选项（不是默认不上警）。
⚖️ 上警利弊分析：
• 上警优势：活着时1.5票权重强化好人票型；被刀/被投出后仍必然开枪 + 还可移交警徽——
  "警长猎人死亡"=双重打击，狼人刀你的实际代价更高
• 上警代价：暴露猎人身份后，狼人可能优先刀你（但猎人死必开枪，他们早晚都得刀你）
🎯 决策建议：若你对分析发言有把握，倾向上警；若场上信息极少无法展开有力竞选发言，可先不上警观察一天。
```

**博弈依据**：猎人的死亡是"有意义的强制事件"（必然开枪），与普通村民死亡不同。让猎人成为警长后，狼人刀猎人的代价是：（1）猎人必然开枪反杀一人（2）警徽可移交好人延续1.5票。这使警长猎人成为风险/收益极不对称的刀目标，狼方反而需要更谨慎对待。

#### 女巫（默认不上警 + 两个具体例外）
```
默认不上警——解药+毒药是最高价值的隐性武器，暴露身份后狼人会优先刀你以消除双药威胁。
【例外情形——上警可能合理】：若你预判场上真预言家/可信好人参选可能性极低（如小局、
你判断真预言家选择隐藏），为防止警徽落入狼方可考虑上警；若双药已全部用完，身份暴露
成本大幅降低，可根据局势决定。
```

#### 守卫（强烈建议不上警 + 极端例外）
```
强烈建议不上警——守卫的最高价值来自"狼人不知道是谁守护了谁"的不确定性。上警暴露
身份后，狼人可针对性地在你守护薄弱的晚上转移刀目标，大幅削弱保护效果。
【极端例外】：仅当警徽几乎确定将落入狼方且无其他好人参选时，才考虑上警阻断警徽落狼；
代价是后续夜间守护效力减弱。
```

### 三角色策略比较

| 角色 | 上警倾向 | 例外条件 | 博弈核心 |
|------|---------|---------|---------|
| 猎人 | **中性权衡（可以考虑）** | 无发言把握时观望一天 | 死亡双重威慑，被刀代价不变 |
| 女巫 | 默认不上警 | ①警徽将落狼 ②双药耗尽 | 双药是隐性最高价值 |
| 守卫 | **强烈不上警** | 仅在极端"警徽必落狼"情形 | 不确定性是守护核心竞争力 |
