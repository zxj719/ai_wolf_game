**为什么狼人优先级最高**：
1. 狼人 identity_table 最复杂（战略私有注记：高优先刀口、公开叙事维持）
2. 多步思维框架（Step1-Step5），Step 0 引导读取后能显著提升 Step2 威胁识别质量
3. NIGHT_WOLF 已有成熟 Step 0 模式，DAY_SPEECH 缺少对称读取是明显的非对称漏洞
4. 狼方策略一致性是 AI 对局游戏观赏性的核心决定因素

### 狼人 DAY_SPEECH 新增 Step 0

**修复位置**：`ROLE_DAY_SPEECH_PROMPTS['狼人']` 的 `【思维框架（在 thought 中完成）】` 段，Step1 之前插入 Step0。

**新增内容**：
```
Step0: 【读取跨轮威胁积累（D2+适用；首日无历史可跳过）】查看【你之前的身份推理表】：
       reason 含"高优先刀口"的玩家是你跨轮积累的高威胁好人，以此为 Step2 威胁识别的
       起点——评估他们今天的言行是否改变了威胁等级；通过话题引导和投票方向逐步配合夜间
       刀目标计划，保持白天言行与夜间战略的一致性，而非每轮从头评估威胁。
```

**博弈论依据**：
- Wang 2025 信息链理论：多轮行为一致性是识别/扮演狼人的核心能力
- 白天发言中的话题引导和投票方向应与夜间刀口计划协调，而不是每轮临时决策
- 读写关键词对齐：Step 0 使用 "高优先刀口" 与 identity_table 写指导的关键词精确对齐

**关键设计决策（R1/R30 白熊效应检查）**：
- "不要主动针对" → 改为 "通过话题引导和投票方向逐步配合" (正向框架，不列具体禁止行为)
- "高优先刀口" 在此处是读取键（search key），非行为禁止词，不触发白熊效应

   - D2+ 狼人 thought 字段是否出现"根据历史推理表，X号是高优先刀口"等引用历史积累的推理；
