## 2. 本轮优化详情

### 优化1：魔术师 NIGHT_MAGICIAN hasRevealed 利用（item 34）

**问题根因**：
- `hasRevealed` 在 `DAY_SPEECH` roleParams（line 1204）已传递，但 `NIGHT_MAGICIAN` case（line 1265）从未传入 `getMagicianNightActionPrompt`。
- 魔术师白天宣告身份后，夜间行动提示词依然按"身份未暴露"优先级（保核 > 换刀 > 自保）运行。
- 博弈论依据：身份暴露后，狼人在夜间将魔术师列为首选刀口（公开神职=高威胁=优先清除）。自保期望收益大幅上升，应成为最高优先级。

**修复路径（3 文件）**：

| 文件 | 改动 | 说明 |
|------|------|------|
| `aiPrompts.js` NIGHT_MAGICIAN case | +1 行：`hasRevealed: currentPlayer?.hasRevealed` | 调用端接通状态 |
| `magician.js` 参数解构 | +1 项：`hasRevealed,` | 解构新参数 |
| `magician.js` 策略逻辑 | `if (hasRevealed) { … } else { … }` | 身份已公开：P-A=自保，P-B=保核，P-C=换刀；未公开：保持原顺序 |

**关键提示内容（hasRevealed=true 时）**：
```
⚠️ 【身份已公开 — 优先级系统调整】你已宣告魔术师身份，狼人本晚极可能将你列为首选刀口！
✦ 优先级A（自保躲刀，最高）：强烈建议自换以躲避可能的刀
  · 选择一名狼嫌疑人与你交换：狼刀你 → 实际狼嫌疑人死亡
  · 你存活才能继续帮助好人阵营；今晚被刀将彻底失去交换能力
  · 自换目标优先选狼嫌疑人，其次选普通村民（不要换已知神职）
```

### 优化2：骑士终局决斗阈值自适应（item 32/35）

**问题**：骑士提示词固定要求"A级≥70%，B级≥60%"才能决斗。终局（≤5人）每轮淘汰的牌价值大幅提升，高阈值会导致骑士在局势紧迫时仍持牌不出，坐视好人败局。

**修复路径（2 文件）**：

| 文件 | 改动 | 说明 |
|------|------|------|
| `aiPrompts.js` DAY_SPEECH roleParams | +1 行：`aliveCount: players.filter(p => p.isAlive).length` | 传入存活人数 |
| `knight.js` `getKnightDaySpeechPrompt` | `const isEndgame = aliveCount && aliveCount <= 5` + 动态 threshold | 终局降阈 |

**阈值对照**：

| 场景 | 优先级A | 优先级B |
|------|---------|---------|
| 普通（>5人） | ≥70% | ≥60% |
| 终局（≤5人） | ≥50% | ≥40% |

**终局提示文本**：
```
⚠️ 【终局紧急模式（5人存活）】每轮淘汰牌价值极高，优先快速清除狼人！
阈值已自动降低（A级≥50%，B级≥40%）。不能继续等待时机，有决斗把握即应出手。
```
