// R112: 女巫 SHERIFF_SPEECH 专属提示词——信息权威 × 药效执行力框架
const witchSsSaveHave = currentPlayer?.hasWitchSave ?? false;
const witchSsPoisonHave = currentPlayer?.hasWitchPoison ?? false;
let witchSsMedStatus;
if (witchSsSaveHave && witchSsPoisonHave) {
    witchSsMedStatus = '双药均在：...';
} else if (witchSsSaveHave) {
    witchSsMedStatus = '解药在手/毒药已用：...';
} else if (witchSsPoisonHave) {
    witchSsMedStatus = '毒药在手/解药已用：...';
} else {
    witchSsMedStatus = '双药已用：...';
}
const witchSsHint = hasRevealedIdentity ? `已跳身份版...` : `未公开版...`;

## 女巫/守卫警长竞选实际效果（场景举例）

### 场景 A：女巫已跳身份，双药均在
**优化前**：`你是好人（女巫）：竞选发言 = 分析能力展示，说清3点……`

**优化后**：
```
你是女巫（已跳身份）：竞选核心 = "信息权威 × 药效执行力"。
双药均在：竞选核心论点是"双药的主人才是警徽的最佳归宿"——解药+毒药是场上最高价值的未打出资产，1.5票是这两张牌的杠杆放大器。
【竞选3要点】：
1. 药效决策回顾：简述解药或毒药的使用/保留判断（不可伪造的判断力证明）
2. 当前局势判断：基于药效信息，你认为最可疑的是谁
3. 警徽流计划 / 价值主张
```

### 场景 B：守卫身份未公开竞选

**优化前**：`你是好人（守卫）：竞选发言 = 分析能力展示……`

**优化后**：
```
你是守卫（身份未公开）：以"稳健理性的好人"身份竞选，绝不透露守卫身份（避免被针对）。
警长核心价值是引导精准投票——展示冷静局势分析，证明你不会被带偏节奏。
【竞选3要点（不含任何守卫信号）】：
1. 昨夜局势分析（最可疑玩家 + 推理依据，展示理性判断而非情绪反应）
2. 你当选后的票型方向（具体到玩家或方向）
3. 警徽流 / 价值主张
```
