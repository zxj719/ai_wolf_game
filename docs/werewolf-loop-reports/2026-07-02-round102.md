### 新增变量（`src/services/aiPrompts.js` — NIGHT_WOLF case）

在 `wolfNightPrevDay` 声明之后、`wolfNightPeaceStep` 之前插入（R102 专属块）：

```javascript
// R102: NIGHT_WOLF 两连/三连平安夜换刀决策框架（零间接推断）
const isConsecutivePeacefulNightWolf = ctx.dayCount >= 3 && isNightPeacefulWolf &&
    ctx.fullGameTimeline?.includes(`N${ctx.dayCount - 2}:平安夜`);
const wolfNightPrevPrevDay = ctx.dayCount >= 3 ? ctx.dayCount - 2 : 0;
const isTripleConsecutivePeacefulNightWolf = ctx.dayCount >= 4 && isConsecutivePeacefulNightWolf &&
    ctx.fullGameTimeline?.includes(`N${ctx.dayCount - 3}:平安夜`);
const wolfNightThreePrevDay = ctx.dayCount >= 4 ? ctx.dayCount - 3 : 0;
let tripleConsecutivePeaceNightHintWolf = '';
let consecutivePeaceNightHintWolf = '';
if (isConsecutivePeacefulNightWolf) {
    // 三连 hint（前置注入到两连 hint 头部）
    tripleConsecutivePeaceNightHintWolf = isTripleConsecutivePeacefulNightWolf ? `...` : '';
    // 两连 hint（前置注入到 wolfNightPeaceStep 头部）
    consecutivePeaceNightHintWolf = `${tripleConsecutivePeaceNightHintWolf}⭕【两连平安夜...】`;
}
```

### 三层激活条件（超集原则）

| 层级 | 条件 | 行动建议 |
|------|------|---------|
| 单夜 (`isNightPeacefulWolf`) | `dayCount > 1 && lastNightInfo 含平安夜` | 路径A/B 分析守卫/女巫来源 |
| 两连 (`isConsecutivePeacefulNightWolf`) | `dayCount >= 3 && isNightPeacefulWolf && N(dayCount-2):平安夜` | confidence 升 25-35，换次高目标 |
| 三连 (`isTripleConsecutivePeacefulNightWolf`) | `dayCount >= 4 && isConsecutivePeacefulNightWolf && N(dayCount-3):平安夜` | confidence 升 35-45，必须换目标 |

### 三连三路径框架（零间接推断）

```
路径A：三夜刀同一目标 → 守卫极可能固定轮守，换刀 confidence 升 35-45；今晚必须换次高威胁目标
路径B：两夜刀同一目标+一夜不同 → 重复目标连守概率高，换刀 confidence 升 25-35
路径C：三夜各不同目标 → 守卫随机轮守，按单夜路径A/B独立评估
```
