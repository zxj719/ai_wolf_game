## 2. 本轮优化详情

### 问题背景

Round 13 完成了夜间多狼协作（首狼决策人机制）。但白天 `wolfTeammatesHint` 只有三条通用原则（立场分散/投票错位/制造分歧感），没有具体分工建议。

Wang 2025 (arxiv:2408.17177) 的核心发现之一："role differentiation"（角色分化）是多代理欺骗中最关键的成功要素——一旦两狼立场高度统一，好人可以通过相关性检测同时曝光两者；但若一狼主动进攻、另一狼保持中立评委姿态，好人很难从相关性角度识别串供。

### 修改内容

**文件**：`src/services/aiPrompts.js`，`ROLE_DAY_SPEECH_PROMPTS['狼人']` 函数（原 line 940-942）

**改动前**：单行三元表达式，`wolfTeammatesHint` 只包含 3 条通用原则。

**改动后**：`let wolfTeammatesHint` + `if` 块，包含：

```
totalWolves = teammates.length + 1
isLateGame = aliveCount <= totalWolves * 2 + 1

roleDiv:
  2狼 → 主动方（先开口）vs 低调方（后开口）的角色分工
  3+狼 → 激进狼（1名指控好人）vs 分析狼（其余装评委）

lateHint（残局模式）:
  重心从角色声明转为票型精算
  被质疑时抢先指控第三方转移焦点

+ 原有 3 条通用原则保留不变
```

### 博弈论依据

- Wang 2025: role differentiation 是多代理欺骗的核心机制
- 主动方/低调方分工：先发言者拿主动权（established first-mover advantage），后发言者可根据场上反应调整
- isLateGame 阈值 `aliveCount <= totalWolves * 2 + 1`：wolf-parity danger zone 标准定义（再消灭1个好人即触发数量平衡）

### 无效化防御

旧版三条原则（立场分散/投票错位/制造分歧感）全部保留，新的角色分化建议是追加层，不是替换层。这确保回归风险最小。
