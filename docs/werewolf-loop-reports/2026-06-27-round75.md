## 本轮优化：预言家 NIGHT_SEER 夜间查验顺序个性化

### 问题诊断

R74 补完了女巫 NIGHT_WITCH 个性化，R74 的"下轮建议"将预言家 NIGHT_SEER 个性化列为优先级 #1（高优先）。

NIGHT_SEER case 的当前状态：历史查验候选读取（seerHistoryStep）+ 三阶段策略框架（首夜/残局/正常优先级）+ 悍跳警报注入（counterClaimText）均完备，但"如何在框架内选择查验目标"对所有个性类型一视同仁。预言家最核心的策略差异恰恰在于"查验方向选择偏好"——aggressive 玩家直刺最高威胁，cautious 玩家用多元化路径掩护推断方向，cunning 玩家会故意查已知好人制造假象。

### 博弈论依据

Wang 2025 (arxiv:2408.17177) 指出不同认知风格对"信息获取策略"的风险偏好本质不同：

| 个性类型 | 核心特质 | 查验目标选择倾向 |
|---------|---------|--------------|
| aggressive | 主动进攻 | 主动威胁型：直接查最高嫌疑目标，早确认=早带全员集票 |
| cautious | 谨慎稳妥 | 边缘安全型：查边缘/陌生目标，多元化路径掩护推断主方向 |
| logical/analytical | 数据驱动 | 推理优化型：按 identity_table confidence 选信息增量最大目标 |
| cunning | 博弈欺骗 | 情报迷雾型：故意查确认好人，制造"查验方向随机"假象 |
| emotional | 感知驱动 | 直觉导向型：白天强烈直觉感知优先于数据 confidence 值 |
| contrarian | 反直觉 | 反预判型：预判狼人在预判被查目标，选次优但最不被预期的候选 |
| steady | 平衡渐进 | 平衡渐进型：严格按①②③④⑤优先级框架推进，不跳跃偏废 |

### 修复内容（`src/services/aiPrompts.js`，NIGHT_SEER case）


```javascript
const seerNightPersonalityType = currentPlayer?.personality?.type || '';
let seerNightStyle = '';

    seerNightStyle = '\n【你的查验风格】主动威胁型：...';
} else if (seerNightPersonalityType === 'cautious') {
    seerNightStyle = '\n【你的查验风格】边缘安全型：...';
```

**向下兼容**：无个性类型时 `seerNightStyle = ''`，prompt 退化到 R74 状态 ✅。
