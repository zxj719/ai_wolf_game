**LW_WINDOW 预防性升级 5500 → 7000**

```
LW block size: 5286 chars

余量:           214 chars（< 300 chars 安全线）
```

R128 LEARNINGS 铁律：
> 余量 < 300 chars 时预防性升级：将 LW_WINDOW = 5500 → 7000
