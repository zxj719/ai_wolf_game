## 本轮优化：猎人/守卫 personalityLens 注入

### 问题

R67（村民）和 R68（预言家/女巫）已为 DAY_SPEECH 添加个性化发言风格注入，但猎人和守卫仍缺少对应的 `personalityLens`。猎人和守卫有隐藏身份优势（知道自己是神职），其伪装成普通村民的"风格多样性"是最高可观战价值来源之一——但7局猎人发言风格完全相同，无法区分不同性格的猎人。

### 修复内容

**`src/services/aiPrompts.js`**

1. **猎人 personalityLens（R69）**：7种个性类型分支，围绕"如何伪装成不同风格的村民"设计：
   - `logical/analytical`：逻辑推演型（用推理链建立理性村民形象）
   - `aggressive`：强势出击型（第一个指名目标，威慑狼人）
   - `emotional`：感知驱动型（直觉式表达掩护信息优势）
   - `contrarian`：反向质疑型（差异化立场减少针对风险）
   - `cunning`：提问引导型（多问少说，减少立场暴露）
   - `cautious`：精炼谨慎型（有充足依据才开口）
   - `steady`：稳健平衡型（先认可再补充，建立可信分析者形象）

2. **猎人发言字数差异化**：`hunterSpeechLen`
   - aggressive：40-60字（简短攻击性）
   - cautious：60-100字（深思熟虑）
   - 默认：40-80字

3. **守卫 personalityLens（R69）**：7种个性类型分支，围绕"如何隐藏守卫信息、以何种风格融入讨论"设计：
   - `logical/analytical`：逻辑推演型（守护知识转化为可说的推理）
   - `aggressive`：主动出击型（积极追狼转移对守卫身份的关注）
   - `emotional`：共情共鸣型（感性风格融入讨论，信息内含不露）
   - `contrarian`：独立判断型（独立思考者形象减少针对）
   - `cunning`：信息潜伏型（少表态，旁观者角度最大化隐蔽性）
   - `cautious`：低调精炼型（发言精炼，守护信息不由自己泄露）
   - `steady`：平衡协调型（稳健中间人，关键时刻凭公信力推动投票）

4. **守卫发言字数差异化**：`guardSpeechLen`
   - aggressive：35-55字（简短有力）
   - steady：45-75字（从容稳健）
   - 默认：40-70字

5. **roleParams 注释更新**：`R67/R68/R69：个性类型（村民/预言家/女巫/猎人/守卫 DAY_SPEECH 个性化发言风格注入）`

### 注入位置

猎人：`5. 心中锁定` 策略点之后 → `${hunterPersonalityLens}` → `【思维链】`  
守卫：`5. 简短发言` 策略点之后 → `${guardPersonalityLens}` → `【思维链】`

### 白熊效应合规

所有 lens 分支为正向指令（"展现X形象"、"用Y方式表达"），无"不要说"、"禁止"、"绝不能"等负向禁止词 ✅

### Lens 设计原则（R69 新增）

猎人/守卫的 personalityLens 与村民/预言家/女巫有本质差异：  
- 村民 lens = "如何分析"（分析视角差异）  
- 预言家 lens = "如何呈现验证结果"（报验风格）  
- **猎人/守卫 lens = "如何伪装成普通村民"**（伪装风格）

伪装风格的设计要解决猎人/守卫的核心张力：他们知道信息优势，但必须在发言中隐藏神职身份。不同个性产生不同的"信息泄露风险控制策略"。
