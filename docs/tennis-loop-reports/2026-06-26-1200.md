## 三、本轮新增改进（R84 专属）

### 改进 1：FeedbackWidget Skip 状态修复

**问题（UX 缺陷）**

```jsx
// 之前：跳过和提交显示同一消息
<button onClick={() => setDone(true)}>跳过</button>
// → 显示 "✅ 感谢你的评价，会帮我们把游戏做得更好！"
// 用户看到 checkmark + "感谢评价" 却没有实际提交任何内容
```

**修复**

```jsx
// 之后：三状态区分
const [status, setStatus] = useState('idle'); // 'idle' | 'submitted' | 'skipped'

// 提交 → "✅ 感谢你的评价，会帮我们把游戏做得更好！"
// 跳过 → "好的，没问题 🎾"（灰色，中性语气）
```

**新增 CSS**

```css
.tennis-scope .fb-skip-msg {
  text-align: center; color: var(--muted, #a1a1aa); font-size: .88rem; padding: 8px 0;
}
```

| 场景 | 之前 | 之后 |
|------|------|------|
| 用户点「提交评价」 | ✅ 绿色感谢 | ✅ 绿色感谢（不变） |
| 用户点「跳过」 | ✅ 绿色感谢（误导！） | 灰色「好的，没问题 🎾」 |
