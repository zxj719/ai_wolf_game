## 三、本轮改动

### 功能：对手赢盘鼓励提示（`OppSetBanner`）

**背景与动机**

至 R141，对战界面盘级里程碑反馈状态：

| 时刻 | 反馈 |
|------|------|
| 玩家赢下第一盘 | ✅ R139：绿色横幅「🎾 先下一城！第一盘胜」 |
| 对手赢下某盘（非赛末） | ~~无~~（本轮补全） |

玩家处于落后态势时，没有任何来自界面的鼓励信号，只能看到计分板从 `SETS 1-0` 变为 `SETS 1-1` 或 `SETS 0-1`。对家庭玩家而言，这个时刻最需要情绪托底——无论是「追平了，决胜盘加油」还是「扳回来，别放弃」。

**实现细节**

完全对称于 R139 的 `SetWinBanner`，仅改变监听对象（`sets[1]` 代替 `sets[0]`）和文字内容。

```jsx
// BattleScreen.jsx — 与 setWinBanner 紧邻
const prevOppSets = useRef(0);
const [oppSetBanner, setOppSetBanner] = useState(null);
useEffect(() => {
  const cur = state.score.sets[1];
  if (cur > prevOppSets.current && !state.score.matchOver) {
    prevOppSets.current = cur;
    const tied = state.score.sets[0] === cur;
    setOppSetBanner({ tied, key: Date.now() });
    const t = setTimeout(() => setOppSetBanner(null), 1400);
    return () => clearTimeout(t);
  }
  prevOppSets.current = cur;
  return undefined;
}, [state.score.sets[1], state.score.matchOver]);
```

**触发场景与消息**

| 对局态势 | sets 状态 | 显示内容 |
|---------|-----------|---------|
| 对手先赢一盘，玩家落后 | `[0, 1]` | `💪 扳回来！下一盘加油！` |
| 玩家已赢一盘，对手追平 | `[1, 1]` | `💪 追平！决胜盘——绝不放弃！` |

`matchOver=true` 时（对手 2-0 赢赛或玩家 2-1 胜赛最后一盘）不触发——结算由 `ResultScreen` 接管。

**视觉设计**

与 `SetWinBanner` 使用相同的 overlay 架构（`position: fixed; inset: 0; z-index: 86`），但配色改为红系，与失分/被克语义一致：

- 背景：`rgba(20,10,10,.87)` + 红色边框 `rgba(239,68,68,.75)`
- 文字色：`#f87171`（与 `fx-redflash` 同色系）
- 动画：复用相同的弹出曲线 `bt-opp-set-pop`，时长 1.4s

`prefers-reduced-motion` 规则同步补充。
