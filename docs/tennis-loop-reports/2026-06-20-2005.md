## 二、本轮改动

### 问题背景

赛事报名页面（mode 屏）的绝技图鉴信息仅以文字「📖 绝技图鉴 X/7」展示，没有视觉上的芯片/卡片呈现：
1. **移动端不友好**：仅靠 HTML `title` attribute 传递绝技描述，而移动端没有 hover 状态，用户根本看不到说明。
2. **养成感缺失**：玩家不知道自己「还差几个」具体哪些绝技，收集驱动力弱。
3. **发现性差**：新手完全不知道「击败某个家人可以解锁绝技」的机制。

### 修复方案

在 mode 页的成就徽章条和四个模式按钮之间，插入一行 **绝技图鉴芯片组**：

#### 视觉设计

- 7 枚芯片固定展示（对应 7 名家人的绝技），保持固定顺序（与 `ULTIMATES` 定义顺序一致）
- **已解锁**：陶土橙（`--clay` #c25e1d）边框 + 文字，`cursor: pointer`，hover 有背景光晕
- **未解锁**：极低不透明度白色边框，`filter: grayscale(1)`，`disabled` 属性（不可聚焦点击）
- **展开状态**：点击已解锁芯片 → 切换 `expanded` class → 在芯片下方追加一行 `.ult-chip-desc` 描述文字
- **手风琴模式**：同时只能展开一枚（`setExpandedUlt(expanded ? null : name)` 点同一枚则收起）

#### 实现细节

**`src/modules/tennis/TennisRoute.jsx`（+24 行）**

```js
// 新增 state
const [expandedUlt, setExpandedUlt] = useState(null);

// 新增派生值
const unlockedUltNames = new Set([ownUltimate, ...progress.unlockedMoves].filter(Boolean));
```

芯片组 JSX（在成就条之后、`.opts` div 之前）：

```jsx
<div className="ult-gallery">
  <span className="ult-gallery-label">📖 绝技图鉴（点击已解锁绝技查看说明）</span>
  <div className="ult-chips">
    {Object.entries(ULTIMATES).map(([name, u]) => {
      const unlocked = unlockedUltNames.has(name);
      const expanded = expandedUlt === name;
      return (
        <button
          key={name}
          type="button"
          className={`ult-chip${unlocked ? ' unlocked' : ' locked'}${expanded ? ' expanded' : ''}`}
          onClick={() => setExpandedUlt(expanded ? null : name)}
          disabled={!unlocked}
          title={unlocked ? u.desc : `击败 ${u.owner} 解锁`}
        >
          <span className="ult-chip-face">{u.face}</span>
          <span className="ult-chip-name">{name}</span>
          {expanded && <span className="ult-chip-desc">{u.desc}</span>}
        </button>
      );
    })}
  </div>
</div>
```
