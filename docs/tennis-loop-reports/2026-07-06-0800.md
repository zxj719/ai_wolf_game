## 本轮主要任务：体力告急引导（LowEnergyHint）

### 问题分析

R135 报告揭示了策略分层数据：

| 策略 | noob 胜率 | avg 胜率 |
|------|-----------|----------|
| random | 9.0% | — |
| energySaver | **4.3%** | 53.8% |
| tellReader | **77.8%** | 100.0% |

`energySaver` 胜率低于 `random` 的根本原因：玩家看到体力下降，开始刻意节省，但所有有效招式都是正耗体，实际反而拖长对局导致更多力竭惩罚。

**游戏中唯一的负耗体招式**：切削（`energyCost: -3`，每出一次回体 3 点）

使用切削的角色：**Elza、Ross、铁蛋、莹**（诚/菲比/丫 无此招）

**当前问题**：玩家能看到体力条变红，但界面上没有明确指向"切削可以回体"的提示。切削的描述"低平切削改变节奏，顺势喘口气"虽有语义提示，但与其他招式同等排列，没有优先级视觉信号。

### 实现

#### 1. `BattleScreen.jsx` — 新增 `LowEnergyHint` 组件

```jsx
function LowEnergyHint({ pEnergy, phase, playerMoves }) {
  if (phase !== 'cards') return null;
  if (pEnergy >= 30) return null;
  if (!playerMoves?.includes('slice')) return null;
  const tier = pEnergy < 15 ? 'critical' : 'warn';
  return (
    <div className={`bt-low-energy-hint ${tier}`}>
      <span className="bt-leh-icon">{pEnergy < 15 ? '🚨' : '⚠️'}</span>
      <span className="bt-leh-text">
        体力告急（{Math.round(pEnergy)}）！「{MOVES.slice.name}」耗体 −3 可回体续命
      </span>
    </div>
  );
}
```

**触发条件**：
- `phase === 'cards'`（出招阶段，非小游戏/结算时不干扰）
- `pEnergy < 30`（接近力竭线 20 的危险区）
- `playerMoves.includes('slice')`（仅对有切削的角色显示，避免无效提示）

**两档视觉**：
- `warn`（体力 15-29）：橙色 `rgba(251,146,60)` 警告框
- `critical`（体力 < 15）：红色 `rgba(239,68,68)` + `tennis-bounce` 动画

**展示位置**：位于 `OppCoachHint`（克制指导）和 `CrisisHint`（关键分警报）之间，形成：
1. 读招/克制引导（OppCoachHint）
2. **体力告急提示（LowEnergyHint）** ← 本轮新增
3. 关键分警报（CrisisHint）

#### 2. `tennis.css` — 新增样式

```css
.tennis-scope .bt-low-energy-hint {
  display: flex; align-items: center; gap: 7px;
  padding: 6px 10px; margin-bottom: 9px; border-radius: 8px;
  background: rgba(251,146,60,.10); border: 1px solid rgba(251,146,60,.35);
}
.tennis-scope .bt-low-energy-hint.critical {
  background: rgba(239,68,68,.12); border-color: rgba(239,68,68,.40);
  animation: tennis-bounce 1.0s ease infinite;
}
.tennis-scope .bt-leh-text { font-size: .68rem; color: rgba(251,146,60,.95); font-weight: 600; }
.tennis-scope .bt-low-energy-hint.critical .bt-leh-text { color: rgba(252,165,165,.95); }
```

颜色语义与现有系统一致：
- 绿色系（bt-fmh/coach-balanced）→ 均势/信息性
- 青色系（bt-coach-hint）→ 克制技巧
- 黄绿系（bt-crisis/tell）→ 读招/关键分
- **橙红系（bt-low-energy-hint）→ 体力资源紧急** ← 本轮新增
