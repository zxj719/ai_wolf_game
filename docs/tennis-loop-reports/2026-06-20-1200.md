## 二、本轮改动

### 问题背景

家族挑战是 6 站梯度连战，连胜的成就感是最大吸引力之一——但当前 UI 对「已连胜几次」完全没有显式反馈，玩家只能数进度条圆点。缺少即时的「此刻你有 N 连胜」心理强化。

### 修复方案

**两处展示，含义一致但计数逻辑略有差异（源于 ladderReducer stage 语义）：**

#### `fighting` 状态：进度条中的连胜提示

`ladder.stage` 在 `MATCH_WON` 后并不立即递增（在 `INTERMISSION` 后的 `advanceStage` 才推进），因此在 `fighting` 状态时：
- `ladder.stage` = 正在进行的这一站（0-indexed）
- 已完成的连胜数 = `ladder.stage`（第 0 站时 0 连胜，不显示）

```jsx
{ladder.stage > 0 && <span className="ladder-streak">🔥 {ladder.stage}连胜</span>}
```

效果：第 1 站开始后「🔥 1连胜」、第 3 站「🔥 3连胜」……

#### `between` 状态：站点结算标题中的徽章

此时 `ladder.stage` 仍是刚赢完的那站（INTERMISSION 前），连胜数 = `ladder.stage + 1`：

```jsx
<h2>🏟️ 第 {ladder.stage + 1} 站拿下！{' '}<span className="ladder-streak">🔥 {ladder.stage + 1}连胜</span></h2>
```

效果：赢完第 1 站后标题变「🏟️ 第 1 站拿下！🔥 1连胜」。

#### CSS（+1 行）

```css
.tennis-scope .ladder-streak {
  display: inline-flex; align-items: center;
  background: var(--clay); color: #fff; border-radius: 999px;
  font-family: var(--mono); font-size: .7rem; font-weight: 700;
  padding: 2px 10px; letter-spacing: .04em; white-space: nowrap; margin-left: 8px;
}
```

使用主题色 `var(--clay)`（橙色 `#e2814f`），与进度条「当前站」高亮色一致，视觉上强化「你正在打破纪录」的感觉。

### 用户体验前后对比

| 场景 | 改动前 | 改动后 |
|------|--------|--------|
| 家族挑战第 3 站开始前 | 进度条 3 个圆点高亮，无胜场计数 | 「🔥 2连胜」橙色徽章出现在进度条末尾 |
| 第 2 站赢完赛间页 | 标题「🏟️ 第 2 站拿下！」 | 「🏟️ 第 2 站拿下！🔥 2连胜」 |
| 第 1 站开始前 | 无（0 连胜，badge 隐藏） | 无变化（0 连胜不显示，避免噪音） |
| 球王 6 站全胜结算 | 只有「家族球王加冕！」 | 进度条「🔥 5连胜」 → 结算页已另有 👑 奖励 |
