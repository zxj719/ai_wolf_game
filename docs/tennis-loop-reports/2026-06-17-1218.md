## 二、本轮改动

### 核心功能：SelectScreen 个人历史最佳纪录卡

**问题定位：**

`ladderBestStreak`（轮 28）和 `adventureBestFloor`（轮 29）两项个人纪录已经存在且各模式内部展示，但玩家在**报名选角色阶段**完全看不到自己的历史最佳——缺乏「我上次连赢 3 站，这次要打到 4 站」的会话前目标感，降低了开局动力与多周目意愿。

**实现方案：**

**1. PersonalBestCard 组件（SelectScreen.jsx）**

```jsx
function PersonalBestCard({ progress }) {
  const streak = progress.ladderBestStreak ?? 0;
  const floor = progress.adventureBestFloor ?? 0;
  if (streak === 0 && floor === 0) return null;  // 新用户无干扰
  return (
    <div className="select-pb-card" role="region" aria-label="个人历史最佳纪录">
      <span className="select-pb-label">🎯 历史最佳</span>
      <div className="select-pb-items">
        <div className="select-pb-item">
          <span className="select-pb-icon">🏆</span>
          <span className="select-pb-mode">家族挑战</span>
          <span className="select-pb-val">连赢 {streak} 站</span>
        </div>
        <div className="select-pb-item">
          <span className="select-pb-icon">🗺️</span>
          <span className="select-pb-mode">奇幻闯关</span>
          <span className="select-pb-val">
            {floor >= 3 ? '已通关 🏅' : `第 ${floor} 章`}
          </span>
        </div>
      </div>
      <span className="select-pb-hint">超越自己！</span>
    </div>
  );
}
```

**显示逻辑：**
- 两项纪录均为 0（新用户）→ 返回 null，完全不渲染
- 任一 > 0 → 展示黄绿色横幅卡片
- adventureBestFloor >= 3 → 显示「已通关 🏅」（比「第 3 章」更有仪式感）
- 位置：规则横幅（rule-strip）下方，与角色选择在同一卡片内

**2. 样式（tennis.css，+29 行）**

延续 `ladder-record-banner` / `adv-record-banner` 的视觉语言：
- 背景：`rgba(217,255,61,.06)` + `rgba(217,255,61,.18)` 边框（比内部横幅更克制）
- 数值：`font-family: var(--mono)` + `color: rgba(217,255,61,.9)`（与其他纪录数字一致）
- 「超越自己！」右对齐斜体浅灰，营造轻松鼓励感而不抢注意力

**3. TennisRoute.jsx（+1 行）**

```jsx
<SelectScreen ... progress={progress} />
```

SelectScreen 函数签名新增 `progress = EMPTY_PROGRESS` 默认参数，保证在任何调用方不传 progress 时也能安全降级。
