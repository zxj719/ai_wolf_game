## 二、本轮改动

### 核心功能：ResultScreen 角色胜场进度行

**问题定位：**

结算屏已有「新成就解锁」（AchievementUnlockCard）和「本场亮点」（HighlightsCard），但缺少一个面向长期收集动机的展示——玩家赢了之后没有清晰的视觉提示「我还差几位家人没攻克」。`progress.charWins` 数据早已在 `onSingleMatchOver` 完整更新（`updateProgress` 在 `dispatch` 之前调用），只需传入并渲染即可。

**实现方案：**

**1. ResultScreen.jsx 改动（+37 行）**

- 顶部引入 `CHARS` from `../gameData`
- 新增 `CharWinsProgressCard({ charWins, playerName })` 函数组件：
  - 计算 `clearedCount`（胜场数 ≥1 的角色数）
  - 渲染 7 个 `cwp-chip`：emoji 大图 + 角色名 + 右上角胜场数徽章
  - 已攻克（wins ≥1）：`cwp-chip--won`（黄绿高亮）
  - 本场使用角色：`cwp-chip--cur`（琥珀描边 + 光晕）
  - 全清时 heading 后接 `cwp-all-badge`「🏅 全能选手！」
  - 未全清时底部 hint 文字引导「用每位家人各赢一场，解锁全能选手成就」
- `ResultScreen` 函数签名增加 `charWins` prop（可选，`undefined` 时组件自动返回 null）
- 在 `<AchievementUnlockCard />` 之后、`<HighlightsCard />` 之前注入 `<CharWinsProgressCard charWins={charWins} playerName={p.name} />`

**2. TennisRoute.jsx 改动（+1 行）**

```jsx
charWins={progress.charWins}
```

**3. tennis.css 改动（+35 行）**

新增 11 个 CSS class（cwp-card / cwp-summary / cwp-all-badge / cwp-row / cwp-chip / cwp-chip--won / cwp-chip--cur / cwp-face / cwp-name / cwp-wins / cwp-hint）

**改动文件：**

| 文件 | 操作 | 说明 |
|------|------|------|
| `src/modules/tennis/components/ResultScreen.jsx` | 修改 | CharWinsProgressCard 组件 + charWins prop（+37 行） |
| `src/modules/tennis/TennisRoute.jsx` | 修改 | 传入 charWins={progress.charWins}（+1 行） |
