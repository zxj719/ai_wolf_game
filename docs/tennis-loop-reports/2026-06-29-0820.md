## 本轮实施：对手连胜徽章（opp-streak-badge）

### 功能描述

在 SelectScreen 对手历史面板（`opp-history-chip`）中，当当前选中玩家对某一具体对手连续获胜 ≥2 场时，该对手卡片底部显示橙色 `🔥N连` 徽章（N = 连胜场数）。

**触发门槛 ≥2 的原因**：单场胜利不具有"连续"的统计意义，≥2 才能向玩家传递"我在该对手面前状态稳定"的信号，避免噪声展示。

### computeOppWinStreak 逻辑

```js
export function computeOppWinStreak(records, playerName, oppName) {
  const mine = records.filter((r) => r.p === playerName && r.o === oppName);
  let streak = 0;
  for (let i = mine.length - 1; i >= 0; i--) {
    if (mine[i].sp > mine[i].so) streak++;
    else break;
  }
  return streak;
}
```

- 双维度过滤（player + opponent），不受其他对手记录影响
- 从末尾向前遍历，遇到非胜即断，确保"连续"语义
- sp === so（平局）视为非胜，截断连胜
