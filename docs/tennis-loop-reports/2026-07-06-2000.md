## 三、本轮改动

### 功能：被克制对称红色视觉（`fx-redflash` + `counter-lose`）

**背景与动机**

R137 为「效果拔群」（玩家克制对手）场景引入了绿色全屏脉冲 `fx-greenflash` + `counter-win` 绿色文字高亮。但被克制场景（`r.counterMul < 1 && !r.win`）仅有橙红文字 `fx-burst.bad`，缺乏全屏视觉冲击，导致「赢了绿光闪」与「输了无反馈」的不对称体验。

本轮在三处完成对称：

#### 1. FxOverlay — fx 触发器加 `redFlash: true`

```js
// before
else if (r.counterMul < 1 && !r.win) setFx({ key: Date.now(), type: 'bad', text: '被克制……' });

// after
else if (r.counterMul < 1 && !r.win) setFx({ key: Date.now(), type: 'bad', text: '被克制……', redFlash: true });
```

#### 2. FxOverlay — render 层加 `fx-redflash` div

```jsx
{fx.gold && <div className="fx-goldflash" />}
{fx.greenFlash && <div className="fx-greenflash" />}
{fx.redFlash && <div className="fx-redflash" />}   {/* 新增 */}
<div className={`fx-burst ${fx.type}`}>{fx.text}</div>
```

#### 3. RallyLog — 文本行加 `counter-lose` class

```jsx
// before
<p className={`...${r.counterMul > 1 && r.win && !r.hawkeyeSaved ? ' counter-win' : ''}`}>

// after
<p className={`...${r.counterMul > 1 && r.win && !r.hawkeyeSaved ? ' counter-win' : ''}${r.counterMul < 1 && !r.win ? ' counter-lose' : ''}`}>
```

**新增 CSS（tennis.css）**

```css
/* 动画关键帧 */
@keyframes tennis-redflash {
  0% { opacity: 0; } 15% { opacity: .85; } 100% { opacity: 0; }
}
@keyframes counter-dim {
  0% { background: rgba(239,68,68,.18); }
  100% { background: rgba(239,68,68,.05); }
}

/* 全屏脉冲（镜像 fx-greenflash，用红色椭圆渐变） */
.tennis-scope .fx-redflash {
  position: fixed; inset: 0; z-index: 79; pointer-events: none;
  background: radial-gradient(ellipse at center, rgba(239,68,68,.28), transparent 65%);
  animation: tennis-redflash .9s ease forwards;
}

/* RallyLog 文本行高亮（镜像 counter-win，用红色） */
.tennis-scope .bt-rallylog .counter-lose {
  background: rgba(239,68,68,.05); border-left: 3px solid rgba(239,68,68,.4);
  border-radius: 0 5px 5px 0; padding: 4px 8px;
  animation: counter-dim .7s ease forwards;
}
```

**reduced-motion 更新**

```css
/* 扩展至 fx-redflash + counter-lose */
.tennis-scope .fx-burst, .tennis-scope .fx-goldflash,
.tennis-scope .fx-greenflash, .tennis-scope .fx-redflash { animation: none; opacity: 0; }
.tennis-scope .bt-rallylog .counter-win,
.tennis-scope .bt-rallylog .counter-lose { animation: none; }
```

**视觉颜色选择**

| 场景 | 颜色 | CSS 值 | 透明度策略 |
|------|------|--------|-----------|
| 效果拔群（赢） | 绿 | `rgba(74,222,128,.38)` | 全屏 .38 → 消散 |
| 被克制（输） | 红 | `rgba(239,68,68,.28)` | 全屏 .28（略弱于绿，负面反馈不应强过正面） |

红色脉冲有意调低透明度至 .28（vs 绿色 .38），避免负面反馈在视觉上超过正面——玩家被克制时获得清晰警告，而不是被「惩罚感」淹没。
