## 三、本轮改动

### 功能：Sprint 个人月最高（结算屏）

**背景与动机**

R135 的效率子榜展示了家庭内部同日竞争；但如果单人独玩，效率排名不会显示（`length >= 2`）。「个人月最高」解决了这个问题：即使是独自游玩，玩家也能看到「本月我最好成绩是 X 分」，追踪自己一个月内的成长曲线，而不需要其他家庭成员在场。

家族榜按 pts 全局排（谁打的多）；个人月最高按玩家名过滤后取本月最优——两个维度互补：一个横向（家庭竞争），一个纵向（个人成长）。

**架构设计（纯函数 + 可测试注入）**

`sprintScores.js` 新增两个函数：
- `isThisMonth(ts, { today? })` → `boolean`：判断 `ts` 是否属于当月（同年同月），与 `isToday` 同一注入模式
- `getPersonalMonthlyBest(hiscores, playerName, { today? })` → `Object|null`：过滤当月 + 指定玩家 → 按 pts 降序 + ts 升序取最优

`SprintScreen.jsx` 变更：
- 导入 `getPersonalMonthlyBest`
- `done` 阶段：`personalMonthlyBest = getPersonalMonthlyBest(allHiscores, basePlayer.name)`
- `isMonthlyBest = personalMonthlyBest?.ts === currentTsRef.current`
- banner 扩展：在 isEffChamp 行下新增「📅 本月个人新高！」（橙色，仅当 `isMonthlyBest`）
- 新增「📅 {playerName}的本月最高」展示区（家族榜与效率榜之间），`isMonthlyBest` 时橙色高亮背景

**优先级/边界逻辑**

| 情形 | 表现 |
|------|------|
| 本月无任何记录（首次） | 不显示个人月最高区（`personalMonthlyBest === null`） |
| 本月有记录但当前局非最高 | 灰色背景展示参考值（pts + 日期） |
| 当前局 = 本月最高 | 橙色背景 + `isMonthlyBest` banner「📅 本月个人新高！」 |
| 上月有高分、本月才开始 | 只取本月数据，不跨月混入 |
