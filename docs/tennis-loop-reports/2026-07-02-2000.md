## 三、本轮改动

### 功能：ResultScreen「今日打法风格」徽章（play-style-badge）

**背景与动机**

R120 完成了 matchup 信息链全 5 节点 × 2 情景的完整覆盖。信息层已饱和，下一步转向**内容个性化**：让每局比赛在结算屏有独特的"今天你是什么风格"的角色感。这对家庭游戏场景尤为重要——家人们可以互相比较风格（"我是重炮手！你是心理战专家？"）。

**设计原理**

`matchStats.moveUsage` 已记录每个招式的使用次数（`{ flatDrive: 3, slice: 2, dropShot: 4, ... }`）。`MOVES` 里每招都有 `system` 字段（`'power' | 'spin' | 'net' | 'control'`）。聚合步骤：

1. 遍历 `moveUsage`，按 `MOVES[id].system` 累加使用次数 → `sysTotals`
2. 取总出招数 `total`；若 <2 则不显示（场次太少无意义）
3. 找最高 `[topSys, topCount]`，计算占比 `pct = topCount / total`
4. `pct < 0.40` → 均衡战士（四系均衡，无明显偏向）
5. 否则按 `topSys` 映射风格：

| system  | 风格标签   | 图标 | 颜色            |
|---------|-----------|------|-----------------|
| power   | 重炮手    | 🔨   | `rgba(249,115,22,.85)` 橙 |
| spin    | 旋转大师  | 🌀   | `rgba(34,211,238,.85)` 青 |
| net     | 网前刺客  | 🥅   | `rgba(74,222,128,.85)` 绿 |
| control | 心理战专家 | 🧠  | `rgba(244,114,182,.85)` 粉 |
| （均衡）| 均衡战士  | ⚡   | `rgba(167,139,250,.85)` 紫 |

**40% 阈值设计**：8 招里 power 有 flatDrive+smash 2 招，spin 有 topspin+slice 2 招，net 有 volley 1 招，control 有 dropShot+lob+passingShot 3 招。配招集 4 招选法，均衡玩家平均每系 25%。阈值 40% 意味着玩家明显偏用某一系（约超出均衡 60%），信号清晰。

**UI 位置**：插在 `match-stat-grid`（数值格子）之后、`counter-hl`（克制得分高亮）之前，作为统计卡的导语性总结。

## 四、实现路径

### `src/modules/tennis/components/ResultScreen.jsx`

新增 `playStyleBadge` useMemo（+22 行）在 `topMoveEntry` 之后：

```jsx
const playStyleBadge = useMemo(() => {
  if (!matchStats?.moveUsage) return null;
  const sysTotals = {};
  for (const [moveId, count] of Object.entries(matchStats.moveUsage)) {
    const sys = MOVES[moveId]?.system;
    if (sys) sysTotals[sys] = (sysTotals[sys] || 0) + count;
  }
  const total = Object.values(sysTotals).reduce((a, b) => a + b, 0);
  if (total < 2) return null;
  const [topSys, topCount] = Object.entries(sysTotals).reduce((a, b) => b[1] > a[1] ? b : a);
  const pct = topCount / total;
  if (pct < 0.40) {
    return { icon: '⚡', label: '均衡战士', desc: '四系均衡出招，难以被针对', color: 'rgba(167,139,250,.85)' };
  }
  const STYLE_MAP = { ... };
  return STYLE_MAP[topSys] ?? null;
}, [matchStats]);
```

JSX 插入（+8 行，在 `match-stat-grid` div 后）：

```jsx
{playStyleBadge && (
  <div className="play-style-badge" style={{ borderColor: playStyleBadge.color }}>
    <span className="psb-icon">{playStyleBadge.icon}</span>
    <div className="psb-text">
      <span className="psb-label" style={{ color: playStyleBadge.color }}>今日风格：{playStyleBadge.label}</span>
      <span className="psb-desc">{playStyleBadge.desc}</span>
    </div>
  </div>
)}
```
