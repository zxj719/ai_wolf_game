## 二、本轮改动

### 核心功能：ult-chip 点击展开描述

**问题定位：**

ult-strip 展示家族成员的绝技图鉴（已解锁高亮金色，未解锁灰色）。之前每个 chip 只有 `title` 属性（PC hover tooltip），移动端用户无法查看绝技描述，必须点进备战页才能看到效果说明——增加了一层额外跳转成本。

**实现方案（2 文件变更）：**

**1. TennisRoute.jsx（+17 行）**

新增 `expandedUlt` state（完全复用 `expandedAch` 成就展开设计模式）：

```jsx
const [expandedUlt, setExpandedUlt] = useState(null);
```

ult-chip 从静态 span 升级为可交互按钮，新增 `ult-chip--exp` class 和 ARIA 属性：

```jsx
const isExp = expandedUlt === ultName;
<span
  className={`ult-chip${isUnlocked ? ' ult-chip--on' : ''}${isExp ? ' ult-chip--exp' : ''}`}
  role="button"
  tabIndex={0}
  onClick={() => setExpandedUlt(isExp ? null : ultName)}
  onKeyDown={(e) => (e.key === 'Enter' || e.key === ' ') && setExpandedUlt(isExp ? null : ultName)}
  aria-expanded={isExp}
>
```

ult-strip 下方条件渲染描述行：

```jsx
{expandedUlt && (() => {
  const u = ULTIMATES[expandedUlt];
  const isUnlocked = ultimateOptions.some((opt) => opt.name === expandedUlt);
  return u ? (
    <p className="ult-chip-tip">
      {u.face} 「{expandedUlt}」— {isUnlocked ? u.desc : `${u.owner}的绝技 · 家族挑战连胜可解锁`}
    </p>
  ) : null;
})()}
```

**2. tennis.css（+4 行）**

```css
/* cursor: default → cursor: pointer（所有 chip 可点击） */
.tennis-scope .ult-chip { ... cursor: pointer; ... }

/* 展开态（紫色主调，区分金色已解锁态） */
.tennis-scope .ult-chip--exp { border-color: #a78bfa; background: rgba(167,139,250,.18); transform: scale(1.12); }
.tennis-scope .ult-chip-tip { font-size: .8rem; color: #c4b5fd; font-style: italic; margin: -4px 0 12px; }
```

**用户体验前后对比：**

| 场景 | 改动前 | 改动后 |
|------|--------|--------|
| PC hover 绝技 chip | title tooltip | 相同 + 点击可固定展开 |
| 移动端点击绝技 chip | 无任何反馈 | 展开一行紫色说明（效果/解锁路径） |
| 点击已展开的 chip | — | 收起描述（toggle） |
| 未解锁绝技 | 只知道归属 | 「XX 的绝技 · 家族挑战连胜可解锁」 |
| 键盘导航（Tab/Enter） | 无 | 完全可访问，`aria-expanded` 语义 |
