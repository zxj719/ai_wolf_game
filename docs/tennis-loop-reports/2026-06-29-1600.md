## 三、本轮改动

### 功能：对手「历史最高连胜」标签

**背景与动机**

SelectScreen 的对手历史面板（opp-history-chip）已有：
- ✓ 已挑战标记
- 推荐/劲敌标签（基于总胜率）
- 胜负比（wins/total）
- 最近 3 局走势点（R99）
- 当前连胜徽章 🔥N连（R100，仅展示末尾连续胜）
- 上次对战相对时间（R101）

缺少的是「历史最高成就」——玩家无法感知「我最好时打这位家人赢过几连」。加入历史最高连胜标签后，当玩家看到「Elza · 最高 4连」而当前连胜只有 1 时，会产生「之前最高赢了她 4 连，要重新追回去」的目标感。这是游戏化设计的「个人记录可追」机制（personal record chasing）。

**设计决策**

| 设计点 | 选择 | 理由 |
|--------|------|------|
| 显示条件 | `best >= 2 AND best > currentStreak` | 避免与当前连胜徽章重复；单次胜利无记录意义 |
| 颜色设计 | 淡青色 `rgba(103,232,249,.55)` | 区别于当前连胜的橙色（🔥），"历史记录"用冷色调 |
| 字体权重 | 600（medium-bold） | 比数据标签（500）更醒目，但不及当前连胜（700）显眼 |
| 算法 | 线性扫描累积 `cur/best` | O(n) 单次遍历，无额外排序 |

**关键算法（线性扫描）**

```
当前连胜（computeOppWinStreak）= 从末尾向前计连续胜场
历史最高（computeOppBestWinStreak）= 全程扫描，任意时段的最大连续胜场
```

两者独立计算，语义不同：
- 当前连胜 = "现在状态热不热"（情绪化，当前直觉）
- 历史最高 = "我最好时的成绩"（成就化，长期目标）

**展示场景**

| 场景 | 结果 |
|------|------|
| 历史最高 < 2 | 不显示 |
| 历史最高 ≥ 2 且 = 当前连胜 | 不显示（当前连胜徽章已够） |
| 历史最高 ≥ 2 且 > 当前连胜 | 显示「最高 N连」 |
| 当前连胜 = 2，历史最高 = 5 | 🔥2连 + 「最高 5连」同时显示 |

## 四、实现路径

### `src/modules/tennis/localBoard.js`
新增 `computeOppBestWinStreak(records, playerName, oppName)` 导出函数：

```js
export function computeOppBestWinStreak(records, playerName, oppName) {
  const mine = records.filter((r) => r.p === playerName && r.o === oppName);
  let best = 0, cur = 0;
  for (const r of mine) {
    if (r.sp > r.so) { cur++; if (cur > best) best = cur; }
    else cur = 0;
  }
  return best;
}
```

### `src/modules/tennis/__tests__/localBoard.test.js`
- 导入 `computeOppBestWinStreak`
- 新增 8 个测试用例（`describe('computeOppBestWinStreak')`）：
  - 空记录 → 0
  - 玩家名不匹配 → 0
  - 对手名不匹配 → 0
  - 单场胜利 → 1
  - 单场失败 → 0
  - [W,W,L,W] 最佳为 2（不取末尾 1）
  - [L,W,W,W,L,W] 最佳为 3
  - 混入其他玩家/对手 → 只统计指定组合

### `src/modules/tennis/components/SelectScreen.jsx`
- 导入 `computeOppBestWinStreak`
- 新增 `oppBestStreakMap` useMemo（依赖 `records` + `picked`）：
  - `picked` 为空时返回 `{}`
  - 否则遍历 CHARS，为每个对手存历史最高连胜（只存 ≥ 2 的）
- opp-history-chip 内当前连胜徽章之后，条件渲染历史最高标签：
  ```jsx
  {oppBestStreakMap[c.n] >= 2 && oppBestStreakMap[c.n] > (oppStreakMap[c.n] ?? 0) && (
    <span className="opp-best-streak" aria-label={`历史最高连胜${oppBestStreakMap[c.n]}场`}>
      最高 {oppBestStreakMap[c.n]}连
    </span>
  )}
  ```

### `src/modules/tennis/tennis.css`
新增 `.opp-best-streak`：淡青色 `rgba(103,232,249,.55)`、`.50rem`、居中、`margin-top: 3px`
