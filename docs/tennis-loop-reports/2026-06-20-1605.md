## 二、本轮改动

### 问题背景

玩家打完比赛拿到装备后，下一局从「报名处」重新开始——但 SelectScreen 完全没有装备信息，用户需要先选完角色、经过反应测试、到备战页才能看到装备效果。这造成「我记得上局赢了一件史诗球拍，但我不知道现在有没有带」的记忆负担，也削弱了装备系统的养成感知。

### 修复方案

在 `SelectScreen` 的角色名片格（`.roster`）和规则条（`.rule-strip`）之间，插入一行 **5 槽装备迷你栏**：

#### 视觉设计

- 5 个装备槽依次排列：`🎾 球拍 / 👟 球鞋 / ✋ 手胶 / 💪 护腕 / 🍀 幸运挂件`
- **已装备**：边框颜色 = 品质颜色（普通灰 / 精良蓝 / 史诗紫 / 传说金），图标颜色同步，右下角显示品质首字母（`普`/`精`/`史`/`传`）
- **未装备**：边框 + 图标为低不透明度白色，hover 有 `title` 说明「空」
- **完全无装备**时迷你栏仍展示（5 个灰色空槽），让新玩家知道「装备系统存在，正在等待解锁」

#### 实现细节

**`src/modules/tennis/components/SelectScreen.jsx`（+20 行）**

- 新增导入：`EQUIPMENT_SLOTS, SLOT_META, RARITY_META` from `../meta/equipment`
- 函数签名加 `equipment = {}` 参数（默认空对象，向后兼容）
- 在 `.roster` 和 `.rule-strip` 之间插入 `.equip-mini-bar` 结构：
  ```jsx
  <div className="equip-mini-bar">
    <span className="equip-mini-label">🎒 当前装备：</span>
    {EQUIPMENT_SLOTS.map((slot) => {
      const item = equipment[slot];
      const color = item ? RARITY_META[item.rarity].color : 'rgba(242,238,224,0.2)';
      return (
        <span key={slot} className="equip-mini-chip"
          style={{ borderColor: color, color: item ? color : 'rgba(242,238,224,0.3)' }}
          title={`${SLOT_META[slot].name}（${item ? RARITY_META[item.rarity].name : '空'}）`}
        >
          {SLOT_META[slot].icon}
          {item && <span className="equip-mini-rarity">{RARITY_META[item.rarity].name[0]}</span>}
        </span>
      );
    })}
  </div>
  ```

**`src/modules/tennis/TennisRoute.jsx`（+1 行）**

```jsx
<SelectScreen onStart={onStart} toast={toast} boardProps={boardProps} equipment={progress.equipment} />
```

`progress.equipment` 是 `{slot: {rarity, special?}}` 字典，已在 `EMPTY_PROGRESS` 中初始化为 `{}`，无需额外处理。

**`src/modules/tennis/tennis.css`（+6 行）**

```css
.tennis-scope .equip-mini-bar { display: flex; align-items: center; gap: 8px; margin-top: 16px; flex-wrap: wrap; }
.tennis-scope .equip-mini-label { font-size: .78rem; color: var(--muted); font-family: var(--mono); white-space: nowrap; }
.tennis-scope .equip-mini-chip {
  display: inline-flex; align-items: center; gap: 3px;
  border: 1.5px solid; border-radius: 8px; padding: 3px 8px;
  font-size: .85rem; line-height: 1; transition: opacity .15s;
}
.tennis-scope .equip-mini-rarity { font-family: var(--mono); font-size: .62rem; font-weight: 700; }
```

使用 `--muted` 和 `--mono` 延续已有 design token，与 `.eq-bonus`（备战页装备加成标签）视觉语言一致。

### 用户体验前后对比

| 场景 | 改动前 | 改动后 |
|------|--------|--------|
| 新玩家第一次报名 | 看不到装备相关信息 | 5 个灰色空槽提示「装备系统存在」 |
| 赢了一件史诗手胶后再次报名 | 不知道装备在不在 | `✋史` 紫色高亮徽标出现在迷你栏 |
| 装备了传说球鞋 | 只能在备战页才看到 | 报名处已显示金色 `👟传` |
| 5 件全部满装 | 完全感知不到装备价值 | 彩色 5 徽标强化「装备收藏」成就感 |
