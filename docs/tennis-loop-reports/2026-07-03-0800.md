## 三、本轮改动

### 功能：五连胜庆祝动效（StreakBurst）

**背景与动机**

R123 「下轮计划」明确列出「5连胜庆祝动效：首次连胜 5+ 时全屏粒子/闪光（纯 CSS+JS，无外部依赖），家庭竞技情境下仪式感最强的单点改进」。ResultScreen 已存在连胜徽章（streak-gold 动画 pill），但仅是角落一小块，缺乏屏幕级的仪式感反馈。家庭游戏场景中，孩子或家人达成连胜是可分享的瞬间，全屏动效能强化这个情绪峰值。

**设计原则**

1. **无阻拦**：`pointer-events: none` overlay，用户仍可正常点击按钮
2. **自消失**：3.5s 后 React state → false，DOM 自动移除，无需用户操作
3. **无障碍支持**：`@media (prefers-reduced-motion: reduce)` 下 `display: none`，前庭障碍用户零干扰
4. **确定性粒子**：`seedParticles()` 使用固定公式（`(7 + i * 3.1) % 100`），不依赖 `Math.random()`，可测试、可复现
5. **纯 CSS 动画**：两个 `@keyframes`（`confettiFall` + `burstLabel`），inline styles 只传 delay/duration/color 等参数，无 canvas、无外部库

**视觉效果**

- 金色径向渐变背景光晕（3.5s 渐隐）
- 中央大字「🔥🔥🔥 N 连胜！🔥🔥🔥」pop in → 持续 → fade out（scale+opacity 动画）
- 30 个彩色纸屑粒子（圆形+方形交替，8 色调色盘：金/橙/青/粉/绿/紫/焦橙/蓝绿）
  - 每粒固定起始横向位置（分散覆盖 0-100%）
  - 动画延迟 0~1.58s（交错入场）、时长 2~3s（下落不均匀）
  - 旋转通过 `--burst-start-rotate` CSS 自定义属性注入每粒偏移，keyframe 叠加 720deg 旋转
