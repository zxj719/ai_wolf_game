## 三、本轮改动

### 功能：ResultScreen「下次备战提示」（next-prep-hint）

**背景与动机**

R107 在结算屏 ending-hero 卡新增了「对手属性速览」（灰色小字，告知数值）。  
R108 在 PrepScreen 首轮新增了「球探速报」（琥珀色，告知策略方向）。

两者解决了不同场景的信息需求，但存在断裂：
- PrepScreen 告诉玩家「对手体力强，练技巧」——赛前有方向。
- ResultScreen 告诉玩家「对手体力 82 技巧 75 心态 69」——赛后有数据，但没有"所以呢？"。

**本轮补充的「next-prep-hint」正是这个「所以呢？」**：把对手属性数据翻译成下次备战的行动建议，完成 PrepScreen 到 ResultScreen 再到下次 PrepScreen 的闭环。

**视觉设计决策：与 PrepScreen 同款琥珀色**

`.prep-strategy-hint`（PrepScreen）已建立"橙色框 = 备战策略"的视觉语义。本轮 `.next-prep-hint` 复用完全相同的样式参数（`color: var(--clay)`、`background: rgba(217,119,6,.08)`、`border-left: 3px solid rgba(217,119,6,.5)`），让玩家一眼识别：同一套橙色意味着同一套语义——"备战指导"。

| 设计点 | 选择 | 理由 |
|--------|------|------|
| 位置 | `matchStats` 卡尾部（opp-moves-section 之后） | 玩家已看完招式分析，此时看属性建议最自然 |
| 内容 | `🏋️ {name} 今日{dominant}见长（{val}），下次备战可优先加点{advice}` | 与 PrepScreen「球探速报」语气一致；锻炼图标暗示「下次准备」 |
| 样式 | 与 `prep-strategy-hint` 完全一致 | 建立并强化「橙色 = 备战建议」视觉词汇 |
| 是否条件渲染 | 无需条件（始终显示，因为 `o` 始终存在） | ResultScreen 只在游戏结束后挂载，`state.opp` 保证非空 |
| 仅在 `matchStats` 存在时显示 | ✅（位于 `{matchStats && ...}` 内） | 与其他统计信息一致；无 matchStats 时也有 ending-hero 的对手属性速览 |

**用户体验示意**

```
📊 本局统计
  ...（球数/ACE/克制/倍率等）

🧠 你本场偏爱
  💥 重炮 ×4  🌀 旋转 ×2  ✂️ 切削 ×1

🔍 对手最常用招式
  ✂️ 切削 ×5  🏹 挑高 ×3  ⚡ 截击 ×2
  💡 下次多出「旋转」来破解 Elza 惯用的切削

🏋️ Elza 今日技巧见长（88），下次备战可优先加点体力或心态
```

玩家在看完本局招式分析后，看到橙色框，立刻知道「下次备战第一轮选体力或心态加点」——下次进入 PrepScreen 时，他会看到同样橙色的「球探速报」再次确认这个建议，形成记忆加强。

## 四、实现路径

### `src/modules/tennis/components/ResultScreen.jsx`

在 `nextAdvice` useMemo（line 110）之后新增 `nextPrepHint` useMemo：

```javascript
const nextPrepHint = useMemo(() => {
  const { sta, skill, mind, name } = o;
  const dominant = sta >= skill && sta >= mind ? '体力' : skill >= mind ? '技巧' : '心态';
  const val = dominant === '体力' ? sta : dominant === '技巧' ? skill : mind;
  const adviceAttr = dominant === '体力' ? '技巧或心态' : dominant === '技巧' ? '体力或心态' : '体力或技巧';
  return `${name} 今日${dominant}见长（${val}），下次备战可优先加点${adviceAttr}`;
}, [o]);
```

在 `topOppMoves.length > 0` 条件块之后、matchStats `</div>` 之前插入：

```jsx
<p className="next-prep-hint">🏋️ {nextPrepHint}</p>
```

### `src/modules/tennis/tennis.css`

在 `.prep-strategy-hint` 规则之后新增 1 条，样式参数完全复用：

```css
.tennis-scope .next-prep-hint { font-size: .78rem; font-family: var(--mono); color: var(--clay); margin: 12px 0 0; padding: 6px 12px; background: rgba(217,119,6,.08); border-radius: 8px; border-left: 3px solid rgba(217,119,6,.5); }
```
