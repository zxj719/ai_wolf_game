## 二、本轮改动

### 改动：连胜徽章双档 CSS 动画

**问题背景**

`ladder-streak` 徽章目前是一枚静态橙色药丸，3 连胜和 5 连胜只有 🔥 数量的文字差异（`🔥`/`🔥🔥`/`🔥🔥🔥`），视觉强度和成就感的差异太弱。玩家达成 5 连胜（家族挑战全程连胜）是极高的成就，需要一个明显的视觉反馈来强化这个里程碑时刻。

**实现方式**

**tennis.css（样式层）**

在 `.ladder-streak` 基础样式之后新增：

```css
@keyframes streakPulse {
  0%, 100% { box-shadow: 0 0 0 0 rgba(226,129,79,0); transform: scale(1); }
  50%       { box-shadow: 0 0 10px 4px rgba(226,129,79,.55); transform: scale(1.10); }
}
@keyframes streakGold {
  0%, 100% { background: var(--clay); box-shadow: 0 0 0 0 rgba(255,204,0,0); color: #fff; }
  50%       { background: #f5b400; box-shadow: 0 0 18px 7px rgba(255,204,0,.60); color: #1a0800; }
}
.tennis-scope .ladder-streak.streak-pulse { animation: streakPulse 1.4s ease-in-out infinite; }
.tennis-scope .ladder-streak.streak-gold  { animation: streakGold .85s ease-in-out infinite; }
```

**动画设计决策**

- `streakPulse`（3-4 连胜）：1.4s 慢脉冲，橙色光晕 + 10% 缩放，醒目但不喧宾夺主
- `streakGold`（5+ 连胜）：0.85s 快频闪烁，背景在 clay→金色（#f5b400）间呼吸，文字色同步从白切黑（对比度保持 ≥4.5:1），金光晕圈 18px 扩散半径
- 两档都用 `ease-in-out` 避免机械感，`infinite` 持续吸睛

**修改位置（JSX 层）**

三处同步修改，均为 modifier class 注入，0 额外状态：

**ResultScreen.jsx**（赛后结算连胜徽章）：
```jsx
<span className={`ladder-streak${winStreak >= 5 ? ' streak-gold' : winStreak >= 3 ? ' streak-pulse' : ''}`}>
```

**LadderScreen.jsx - 对局进行中进度条**（第 2–5 站中 ladder-bar 的 `{ladder.stage}连胜`）：
```jsx
<span className={`ladder-streak${ladder.stage >= 5 ? ' streak-gold' : ladder.stage >= 3 ? ' streak-pulse' : ''}`}>
```

**LadderScreen.jsx - 赛间休息屏标题**（第 2–6 站赛间屏 `{ladder.stage + 1}连胜`）：
```jsx
<span className={`ladder-streak${(ladder.stage + 1) >= 5 ? ' streak-gold' : (ladder.stage + 1) >= 3 ? ' streak-pulse' : ''}`}>
```

**效果映射**

| 场景 | 连胜数 | 视觉 |
|------|--------|------|
| ResultScreen / LadderScreen | 2 | 静态橙色徽章 + 🔥（基准） |
| ResultScreen / LadderScreen | 3–4 | 橙色脉冲动画（glow + scale）+ 🔥🔥 |
| ResultScreen / LadderScreen | 5+ | 金光闪烁动画（clay→金色 循环）+ 🔥🔥🔥 |
| LadderScreen 赛间 | 第 3、4 站后 | streak-pulse（ladder.stage+1 = 3、4） |
| LadderScreen 赛间 | 第 5 站后 | streak-gold（ladder.stage+1 = 5 即最终战前） |
