## 三、本轮改动

### 功能：解说词库扩充（commentary-quip-expansion，CHAR_WIN/LOSS_QUIPS 2→4 条）

**背景与动机**

R122 扩充了对手战后台词（`gameData.js` `CHAR_QUOTES` 3→5 条/状态），R124 加入了五连胜庆祝动效。与 CHAR_QUOTES 不同，`commentary.js` 的 `CHAR_WIN/LOSS_QUIPS` 是**旁白视角**的角色特色解说，混入通用词池（`GENERIC_WIN_LINES`、`CLOSE_LOSS_LINES` 等）中随机选出。每角色仅 2 条，在连续游玩时重复率极高。本轮从 2→4 条，从旁白维度补齐角色鲜活感。

**词池大小影响**

| 胜负类型 | 修复前 pool | 修复后 pool |
|---------|------------|------------|
| 标准胜（2:0 无特殊） | GENERIC(3)+CHAR(2)+SWEEP(3)=8 | GENERIC(3)+CHAR(4)+SWEEP(3)=10 |
| 标准胜（2:1 无特殊） | GENERIC(3)+CHAR(2)+COMEBACK(3)=8 | GENERIC(3)+CHAR(4)+COMEBACK(3)=10 |
| 无条件胜 | GENERIC(3)+CHAR(2)=5 | GENERIC(3)+CHAR(4)=7 |
| 惜败（1:2） | CLOSE(3)+CHAR(2)=5 | CLOSE(3)+CHAR(4)=7 |
| 惨败（0:2） | CRUSHING(3)+CHAR(2)=5 | CRUSHING(3)+CHAR(4)=7 |

注：ACE/CLUTCH/COUNTER 特殊胜负走专属词池，不受影响。

**新增 28 条解说（各角色在性格维度上互补的台词）**

| 角色 | 胜（新增 2 条） | 负（新增 2 条） |
|-----|---------------|---------------|
| 诚 | 挥拍眼神、力量宣言 | 咬牙立誓、加倍复仇 |
| Elza | 棋局预谋、狐步既定 | 默记破绽、棋局未完 |
| 菲比 | 步伐封位、飞弹速度 | 憋泪叫板、飞毛腿宣言 |
| Ross | 胜率已定、反向方程 | 样本迭代、下次精准反制 |
| 铁蛋 | 锅气十足、灶台臂力 | 补仓宣言、功夫未到位 |
| 丫 | 等待时机、让人晃眼 | 尾巴梗、暗记漏洞 |
| 莹 | 诗意弧线、漂亮标准 | 眼眸风云、输得好看 |
