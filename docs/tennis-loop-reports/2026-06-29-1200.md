## 三、本轮改动

### 功能：对手「上次对战」相对时间标签

**背景与动机**

SelectScreen 的对手历史面板（opp-history-chip）已有：
- ✓ 已挑战标记
- 推荐/劲敌标签（基于总胜率）
- 胜负比（wins/total）
- 最近 3 局走势点（R99）
- 连胜徽章 🔥N连（R100）

缺少的是「时间感」——玩家无法感知「我上次打这位家人是多久前」。加入相对时间标签后，当玩家看到「菲比 · 5天前」时会产生「好久没打她了，今天去挑战一下」的冲动，直接驱动复玩动机。这与移动游戏「N 天未登录」提醒的心理机制相同。

**设计决策**

| 设计点 | 选择 | 理由 |
|--------|------|------|
| 数据来源 | `ts` 字段（Unix ms） | R92+ 新增，无需改动记录结构 |
| 过滤维度 | `p === playerName && o === oppName` | 只统计当前选定身份对该对手的记录 |
| 显示时机 | `picked` 已选时才计算 | 未选身份时无法确定「我」的视角 |
| 相对时间格式 | 今日 / 昨日 / N天前 | 精确到天，简洁直观，避免精确到小时引起隐私感 |
| 旧记录（无 ts） | 返回 null，不显示 | 防止显示错误时间；保持老用户数据兼容 |
| 视觉权重 | `.48rem` + `rgba(242,238,224,.32)` 极淡灰 | 时间是辅助信息，不能盖过胜率/标签等核心内容 |

**实现路径**

#### `src/modules/tennis/localBoard.js`
- 新增 `computeOppLastBattleTs(records, playerName, oppName)` 导出函数
- 双维度过滤后取尾部记录的 `ts` 字段（`?? null` 兼容无 ts 字段的旧记录）

```js
export function computeOppLastBattleTs(records, playerName, oppName) {
  const mine = records.filter((r) => r.p === playerName && r.o === oppName);
  if (mine.length === 0) return null;
  return mine[mine.length - 1].ts ?? null;
}
```

#### `src/modules/tennis/__tests__/localBoard.test.js`
- 导入 `computeOppLastBattleTs`
- 新增 7 个测试用例（`describe('computeOppLastBattleTs')`）：
  - 空记录 → null
  - 玩家名不匹配 → null
  - 对手名不匹配 → null
  - 单条有 ts → 返回该 ts
  - 多条 → 返回最后一条 ts
  - 最后一条无 ts 字段 → null
  - 混入其他玩家/对手记录 → 只取匹配组合的最后一条

#### `src/modules/tennis/components/SelectScreen.jsx`
- 导入 `computeOppLastBattleTs`
- 新增 `oppLastBattleMap` useMemo（依赖 `records` + `picked`）：
  - `picked` 为空时返回 `{}`
  - 否则遍历 CHARS，为每个对手存最近一战的 ts
- opp-history-chip 内连胜徽章之后，条件渲染相对时间标签：

```jsx
{oppLastBattleMap[c.n] != null && (() => {
  const diffDays = Math.floor((Date.now() - oppLastBattleMap[c.n]) / 86400000);
  const label = diffDays === 0 ? '今日' : diffDays === 1 ? '昨日' : `${diffDays}天前`;
  return <span className="opp-last-date">{label}</span>;
})()}
```

#### `src/modules/tennis/tennis.css`
- 新增 `.opp-last-date`：`.48rem`、极淡灰（opacity 0.32）、居中、`margin-top: 2px`

**展示场景**

| 场景 | 结果 |
|------|------|
| `picked` 未选 | `oppLastBattleMap = {}`，不渲染任何时间标签 |
| `picked` 已选，该对手有 ts 记录 | 显示「今日」/「昨日」/「X天前」 |
| `picked` 已选，该对手记录无 ts 字段（旧数据） | 不显示（null 过滤） |
| `picked` 已选，从未对战该对手 | 不显示（mine.length === 0） |
