## 二、本轮改动

### 改动：LadderScreen 赛间三选一按钮动态提示

**问题背景**

赛间三选一（特训/按摩/逛店）现有的固定说明文字（「随机属性 +8」「额外 +30 体力」）缺乏上下文感知：体力已 95% 时选按摩几乎无用，但界面上并不体现这一点；越早的站次加练越划算，玩家却需要自己心算。

**实现方式**

在 `LadderScreen.jsx` 中新增模块级纯函数 `getIntermissionHints(ladder)`，在 `between` 渲染分支调用（非 Hook，可条件调用）：

```js
function getIntermissionHints(ladder) {
  const { lastRemainingEnergy, bonusStats, stage } = ladder;
  const energyBase = Math.min(100, lastRemainingEnergy + 40);
  const energyMassage = Math.min(100, lastRemainingEnergy + 40 + 30);
  const massageDelta = energyMassage - energyBase;
  const totalBonus = bonusStats.sta + bonusStats.skill + bonusStats.mind;
  const stagesLeft = STAGE_COUNT - 1 - stage;

  // 特训：剩余站数多→越早加练效益越高
  let trainHint;
  if (stagesLeft >= 4) trainHint = `还剩 ${stagesLeft} 站，加练效益最高`;
  else if (stagesLeft === 1) trainHint = '最后一站，属性直接影响决赛';
  else trainHint = totalBonus < 16
    ? `加成仅 +${totalBonus}，现在加练收益高`
    : `当前加成 +${totalBonus}，看情况`;

  // 按摩：展示具体体力数字，收益低时预警
  let massageHint;
  if (massageDelta <= 5) massageHint = `体力 ${energyBase}%，按摩仅 +${massageDelta}%，收益低`;
  else if (energyBase < 55) massageHint = `体力仅 ${energyBase}%，按摩→${energyMassage}%，推荐`;
  else massageHint = `体力 ${energyBase}%→${energyMassage}%，中等收益`;

  // 逛店：站数少时强调「最后机会」
  let shopHint;
  if (stagesLeft >= 4) shopHint = `还有 ${stagesLeft} 站，装备早买早收益`;
  else if (stagesLeft === 1) shopHint = '最后一次备战，有用就买';
  else shopHint = '灵活选择，升卡最划算';

  return { trainHint, massageHint, shopHint };
}
```

关键设计点：
- `energyBase` 用 `min(100, lastRemainingEnergy + 40)` 而非 `ladder.carryEnergy`，因为 `between` 状态下 `carryEnergy` 是上一战的旧值，`advanceStage` 才会更新它
- `stagesLeft = STAGE_COUNT - 1 - stage`（含即将开始的下一站）作为「加练收益倍数」
- 函数置于模块级（非 Hook），在 `between` 分支内直接调用，合法

JSX 中每个按钮的 `.fx` 之后追加 `.opt-hint`：
```jsx
<span>加练特训<span className="fx"><em>随机属性 +8</em></span><span className="opt-hint">{hints.trainHint}</span></span>
```

CSS 新增一条规则：
```css
.tennis-scope .opt .opt-hint { display: block; font-size: .7rem; color: var(--muted); font-family: var(--mono); margin-top: 3px; opacity: .75; }
```

**示例提示组合（第 1 站胜后，体力剩 30%）**

| 选项 | 提示 |
|------|------|
| 💪 特训 | 还剩 5 站，加练效益最高 |
| 💆 按摩 | 体力仅 70%，按摩→100%，推荐 |
| 🛒 逛店 | 还有 5 站，装备早买早收益 |
