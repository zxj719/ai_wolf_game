## 二、本轮改动

### 核心功能：今日一战（Daily Challenge）

**设计动机：**

上轮报告指出，待部署后数据积累之前，最迫切的是提升**用户粘度**。每日挑战是经过验证的游戏化留存机制：每天固定一个有意义的目标，让玩家有理由"明天还要来"。

**技术实现：**

| 层级 | 实现 |
|------|------|
| 种子 RNG | 日期字符串 → djb2 哈希 → LCG（1664525×s + 1013904223 mod 2³²）→ 确定性整数序列 |
| 对手选取 | 从当前玩家以外的 6 位家人中按种子索引选一人，同一天全体玩家对阵同一家人 |
| 属性范围 | sta/skill/mind 各自在 [45, 85] 之间，略高于普通对手下限（40），给日常挑战加点质感 |
| 完成追踪 | localStorage key `tennis_daily_YYYY-MM-DD`，UTC 日期，跨子夜自动失效 |
| 额外奖励 | 获胜 +30 金币（标准 +25），失败无惩罚，遥测 mode 标注为 `'daily'` |

**Reducer 新 action（SET_DAILY_MODE）：**

```javascript
case 'SET_DAILY_MODE': {
  const dailyFoe = CHARS.find((c) => c.n === action.oppName);
  return {
    ...state,
    mode: 'single',
    screen: 'react',
    opp: { name: dailyFoe.n, face: dailyFoe.f, ...action.oppStats },
  };
}
```

原子操作：一次 dispatch 同时替换对手并进入反应测试，不产生中间渲染帧。

**UI 表现：**

- 未完成：金色边框按钮（`.opt--daily`），显示"今天对阵：🦢 莹（明日更换）"
- 已完成：虚线灰显容器（`.opt--daily-done`），显示"今日一战 · 已完成 · 明天 UTC 0 时重置"
- 胜利 Toast：「🌟 今日一战完成！+30 额外金币 · 掉落 ...」
