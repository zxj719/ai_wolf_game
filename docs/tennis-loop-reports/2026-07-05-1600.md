## 本轮改动：今日一战连续天数追踪（Streak）

### 背景

今日一战（Daily Challenge）已有完成标记与奖励，但没有跨日连续性的追踪。连续 streak 是移动游戏中最有效的日留存机制之一，成本低、效果明确。

### 实现内容

**`src/modules/tennis/meta/dailyChallenge.js`** — 新增 3 个函数：

```js
// 存储格式: tennis_daily_streak → { streak: N, lastDate: 'YYYY-MM-DD' }

loadDailyStreak()           // 返回当前连续天数（0 = 无记录）
updateDailyStreak(today?)   // 幂等更新；注入 today 便于测试
clearDailyStreak()          // 测试辅助
```

逻辑：同一天多次调用不重复计（幂等）；连续日 streak+1；有间隔则重置为 1。

**`src/modules/tennis/TennisRoute.jsx`** — 每日获胜时调用 `updateDailyStreak()`，3天/7天触发里程碑 toast。

**`src/modules/tennis/components/SelectScreen.jsx`** — 在今日一战 banner 头部显示 streak 徽章（≥2 天显示；≥7 天变橙色 🔥）。

**`src/modules/tennis/tennis.css`** — 新增 `.daily-streak-badge` / `.daily-streak-fire` 样式。

**`src/modules/tennis/__tests__/dailyChallenge.test.js`** — NEW：13 个测试用例覆盖所有边界情形（月/年跨界、空状态、幂等、连续、断链重置、7天积累）。
