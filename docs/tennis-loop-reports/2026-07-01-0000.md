## 三、本轮改动

### 功能：SelectScreen「主力角色」标签

**背景与动机**

现有 roster chip 已有：
- ✓ `✦ 最佳`（金色）= 胜率最高的角色（`findBestChar`，≥2 场 ≥1 胜）
- ✓ `出战 N · 赢 N`（灰色小字）= 出战/胜场统计

「最佳」反映的是战绩表现，但未能反映玩家的行为偏好——一个习惯选诚（10场）但对阵Elza胜率最高的玩家，看到的只是诚「无标签」而 Elza「✦ 最佳」，这与「诚是我的主力」的自我认知存在断裂。

本轮加入「⭐ 主力」标签，填补这一心理断层：
- 「✦ 最佳」= **战绩身份**：我赢得最漂亮的是谁
-「⭐ 主力」= **行为身份**：我最常上场的是谁

两者可以同时出现在同一个 chip 上（当最常玩的角色恰好也是胜率最高时），也可以分别出现在不同 chip 上——后者是更有趣的情形，给玩家一个反思：「我最常选的不是我最擅长的，要不要换换？」

**设计决策**

| 设计点 | 选择 | 理由 |
|--------|------|------|
| 函数名 | `findMainChar` | 与 `findBestChar` 语义对称 |
| 门槛 | `minPlayed = 3`（默认） | 避免 1-2 场的随机选择被误标为「主力」 |
| 同场次平局规则 | 胜场更多者优先 | 同等出战频率时，有更好成绩的角色更可信 |
| 颜色 | 蓝色 `#60a5fa` | 区别于金色「✦ 最佳」；「常规」vs「最优」的冷暖色对比 |
| 字号/字重 | `.6rem / 700`（与 best-char-star 完全一致） | 视觉权重相同，语义平等 |
| 是否与「✦ 最佳」互斥 | 不互斥 | 两者语义不同，允许同时出现（如：诚「✦ 最佳 ⭐ 主力」表示玩家最常选诚且战绩也最好） |

**展示场景**

| 场景 | 结果 |
|------|------|
| 无任何角色出战 ≥3 场 | 不显示 `⭐ 主力` |
| 诚出战 5 场（最多），且 ≥3 场 | 诚 chip 显示 `⭐ 主力` |
| 诚 5 场胜率最高 + 诚 5 场出战最多 | 诚 chip 显示 `✦ 最佳` 和 `⭐ 主力`（两行） |
| 诚 8 场（最多出战）+ 菲比胜率最高 | 诚「⭐ 主力」，菲比「✦ 最佳」（分开提示） |

## 四、实现路径

### `src/modules/tennis/localBoard.js`

新增 `findMainChar(charStatsMap, minPlayed = 3)` 导出函数（插入在 `findBestChar` 之后）：

```js
export function findMainChar(charStatsMap, minPlayed = 3) {
  let mainName = null, maxPlayed = 0, maxWon = 0;
  for (const [name, { played, won }] of Object.entries(charStatsMap)) {
    if (played < minPlayed) continue;
    if (played > maxPlayed || (played === maxPlayed && won > maxWon)) {
      maxPlayed = played;
      maxWon = won;
      mainName = name;
    }
  }
  return mainName;
}
```

### `src/modules/tennis/__tests__/localBoard.test.js`

导入 `findMainChar`；新增 `describe('findMainChar')` 6 个用例：
1. 空 map → null
2. 所有角色场次低于 minPlayed → null
3. 单个满足 minPlayed 的角色被选中
4. 返回出战场次最多的角色
5. 场次相同时胜场更多者优先
6. 自定义 minPlayed = 1 时单场角色可入选

### `src/modules/tennis/components/SelectScreen.jsx`

- 导入 `findMainChar`
- 新增 `mainChar` useMemo（依赖 `charStatsMap`）：`findMainChar(charStatsMap)`
- roster chip 内 `bestChar` 标签之后，新增条件渲染：
  ```jsx
  {mainChar === c.n && (
    <span className="main-char-tag">⭐ 主力</span>
  )}
  ```

### `src/modules/tennis/tennis.css`

在 `.best-char-star` 之后新增 1 条规则：
```css
.tennis-scope .roster .chip .main-char-tag { display: block; font-size: .6rem; color: #60a5fa; font-weight: 700; margin-top: 2px; letter-spacing: .04em; }
```
