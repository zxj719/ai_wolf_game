## 三、本轮改动

### 功能：topRally 持久化到本地战绩 + 角色最佳一击展示

**背景与动机**

R144 激活了 `MatchReplayCard`，玩家首次能在结算屏看到本场最佳一击（如「1.72× flatDrive vs topspin」）。但这个信息是「一次性」的：关闭结算屏即消失，没有任何积累性。

家庭游戏的核心乐趣之一是**比较与炫耀**——「我以菲比打出的最高倍率是 1.73×」「诚今天最佳一击比昨天高了！」。将 topRally 持久化到 localBoard，让每个角色的最佳一击成为可积累的「个人 PB」，直接服务于这一场景。

**数据流变化**

```
battleReducer.matchStats.topRally（已有，R144）
    ↓
ResultScreen.saveLocalRecord（本轮新增 topRally 参数）
    → localRecord.tbr: { mv, mult, ctr, cl }  ← 压缩存储
    ↓
SelectScreen.computeTopRallyByChar（本轮新增）
    → 按角色取历史最高 mult 的 tbr
    → roster chip 展示 ✨ {mult}×（CLUTCH 加金色光晕）

TennisRoute.onSingleMatchOver（今日一战路径）
    → saveDailyStats（本轮新增 topRally 参数）
    → myDailyStats.topRally: { mv, mult, ctr }
    → SelectScreen 今日一战统计区展示 ✨ 最佳一击 {mult}×
```

**`localBoard.js` 变更**

1. `saveLocalRecord` 新增 `topRally` 参数：
   ```js
   if (topRally?.pMultiplier > 0) {
     rec.tbr = {
       mv: topRally.pMove,
       mult: +topRally.pMultiplier.toFixed(2),
       ctr: topRally.counterMul > 1,
       cl: !!topRally.clutch,
     };
   }
   ```
   - 压缩至 4 个字段（原始 topRally 对象有 11 个字段）
   - 旧记录无 `tbr` 字段，`computeTopRallyByChar` 自动跳过，向前兼容

2. 新增 `computeTopRallyByChar(records)`：按角色名取所有含 `tbr` 记录中 `mult` 最高的那条，返回 `{ [charName]: { mv, mult, ctr, cl } }`

**`dailyChallenge.js` 变更**

`saveDailyStats` 新增 `topRally` 参数，存储为 `{ mv, mult, ctr }` 或 `null`：
```js
topRally: (topRally?.pMultiplier > 0)
  ? { mv: topRally.pMove, mult: +topRally.pMultiplier.toFixed(2), ctr: topRally.counterMul > 1 }
  : null,
```

**`ResultScreen.jsx` 变更**

```js

  player: p, opp: o, setsP: state.setsP, setsO: state.setsO,
  countersWon: matchStats?.countersWon ?? 0,

// 修改后

```

**`TennisRoute.jsx` 变更**

```js

```

**`SelectScreen.jsx` 变更**

1. 导入 `computeTopRallyByChar`
2. 新增 `const topRallyByChar = useMemo(() => computeTopRallyByChar(records), [records])`
3. Roster chip 新增展示：
   ```jsx

     <span
       className={`char-best-rally${topRallyByChar[c.n].cl ? ' cbr-clutch' : ''}`}
       title={`以${c.n}出战时最佳一击：${topRallyByChar[c.n].mult}×${...}`}
     >
       ✨ {topRallyByChar[c.n].mult}×
     </span>
   )}
   ```
4. myDailyStats 面板新增：
   ```jsx

     <span className="my-daily-chip my-daily-chip-best" title="...">
       ✨ 最佳一击 {myDailyStats.topRally.mult}×
     </span>
   )}
   ```
