## 三、本轮改动

### 功能：克制成功专属视觉脉冲

**背景与动机**

现有特效颜色语义：
- 🟡 金色（`fx-burst.ace` + `fx-goldflash`）→ ACE
- 🟠 橙色（`fx-burst.clutch`）→ 绝技/CLUTCH
- ⚪ 白色（`fx-burst.super`）→ 鹰眼改判 + 克制成功（共用，无区分）
- 🔴 红橙（`fx-burst.bad`）→ 被克制

问题：克制成功是游戏最核心的正向强化时刻（"读招→克制→奖励"），却和鹰眼改判共用白色无差别视觉，无法强化玩家的学习联结。

绿色是此系统中既有的"克制"语义色（`bt-vs .pill.win` 绿底、`bt-rt-dot.won` 绿点），本轮将其扩展到 FX 层。

**实现细节**

#### 1. `BattleScreen.jsx` — `FxOverlay` 类型分离

```jsx
// 之前
else if (r.counterMul > 1 && r.win)
  setFx({ key: Date.now(), type: 'super', text: '效果拔群！' });

// 之后
else if (r.counterMul > 1 && r.win)
  setFx({ key: Date.now(), type: 'counter', text: '效果拔群！🎯', greenFlash: true });
```

新增 `greenFlash` 标志（类比 ACE 的 `gold`），在渲染时触发绿闪背景：

```jsx
{fx.gold && <div className="fx-goldflash" />}
{fx.greenFlash && <div className="fx-greenflash" />}   // 本轮新增
<div className={`fx-burst ${fx.type}`}>{fx.text}</div>
```

#### 2. `BattleScreen.jsx` — `RallyLog` 克制胜增强

```jsx
// 之前
<p className={r.win ? 'res-win' : 'res-lose'}>

// 之后
<p className={`${r.win ? 'res-win' : 'res-lose'}${r.counterMul > 1 && r.win && !r.hawkeyeSaved ? ' counter-win' : ''}`}>
```

克制胜（非鹰眼）的结算文字在基础 `res-win` 上叠加 `counter-win` 样式，排除鹰眼改判（鹰眼已有 `🦅` 图标，两者语义不同）。

#### 3. `tennis.css` — 新增三段样式

```css
/* keyframes */
@keyframes tennis-greenflash {
  0% { opacity: 0; } 15% { opacity: .9; } 100% { opacity: 0; }
}
@keyframes counter-glow {
  0% { background: rgba(74,222,128,.22); }
  100% { background: rgba(74,222,128,.07); }
}

/* FX 绿色体系 */
.tennis-scope .fx-burst.counter { color: #4ade80; letter-spacing: .04em; }
.tennis-scope .fx-greenflash {
  position: fixed; inset: 0; z-index: 79; pointer-events: none;
  background: radial-gradient(ellipse at center, rgba(74,222,128,.38), transparent 65%);
  animation: tennis-greenflash .9s ease forwards;
}

/* RallyLog 克制胜 */
.tennis-scope .bt-rallylog .counter-win {
  background: rgba(74,222,128,.07); border-left: 3px solid rgba(74,222,128,.5);
  border-radius: 0 5px 5px 0; padding: 4px 8px;
  animation: counter-glow .7s ease forwards;
}

/* reduced-motion 更新 */
@media (prefers-reduced-motion: reduce) {
  .tennis-scope .fx-burst, .tennis-scope .fx-goldflash, .tennis-scope .fx-greenflash { animation: none; opacity: 0; }
  .tennis-scope .fx-shake { animation: none; }
  .tennis-scope .bt-rallylog .counter-win { animation: none; }
}
```

**颜色选择依据**

`rgba(74,222,128,.38)` 是 Tailwind `green-400` 透明化版本，与 `pill.win` 背景 `rgba(217,255,61,.55)`（黄绿 `lime-300`）保持同族但不冲突（绿 vs 黄绿），在深色背景下区分度足够。

**视觉层次（从上到下）**

| 时机 | 组件 | 视觉 |
|------|------|------|
| 出招瞬间（counterMul>1 win） | `FxOverlay` | ① 绿色大字 `效果拔群！🎯`（1.1s）② 绿色径向闪光（0.9s） |
| 结算阶段 | `RallyLog.counter-win` | ③ 结算文字绿色左边框 + 0.7s 渐隐光晕 |
| 历史记录 | `RallyTrend.bt-rt-dot.won` | ④ 绿点（已有） |
