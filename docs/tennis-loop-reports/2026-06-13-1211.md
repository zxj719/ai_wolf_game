## 二、本轮改动

### ✅ BattleScreen.jsx — 新增 noviceTell prop

```js
export function BattleScreen({
  player, opponent, deckInstances, ultimate, twists, equip, playerMoves, onMatchOver,
  noviceTell = false,   // ← 新增
}) {
```

BEGIN_RALLY 调度处：

```js
// 新手期（前3局）读招提示强制真实，降低初见系统的困惑感
truthRoll: noviceTell ? 0 : Math.random(),
```

**设计选择：**
- `truthRoll = 0`（绝对确定语义）优于 `Math.random() * 0.74`（引入不必要随机分布）
- `battleReducer` 的 `predictable` twist 已验证"外注 truthRoll=0"路径可行
- prop 默认 `false`，对现有 LadderScreen / AdventureScreen 中的 BattleScreen 调用无任何影响

### ✅ TennisRoute.jsx — 传入 noviceTell

```jsx
<BattleScreen
  player={state.player}
  opponent={state.opp}
  ...
  noviceTell={isNovice()}   // ← 新增
/>
```

`isNovice()` 读 localStorage 的 `tennis_novice_games_v1`，前 3 局返回 true；`incrementNoviceGames()` 在 `onSingleMatchOver` 调用，保证一局内状态稳定不变。
