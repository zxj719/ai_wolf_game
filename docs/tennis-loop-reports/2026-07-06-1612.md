## 三、本轮改动

### 功能：`OppCoachHint + LowEnergyHint → CombinedHint`

**背景与动机**

Cards 阶段（玩家选招前）现有三种提示：

| 提示 | 触发条件 |
|------|---------|
| `OppCoachHint` | 非关键分 + 对手某招已用 ≥2 次 |
| `LowEnergyHint` | cards 阶段 + 体力 <30 + 有切削球 |
| `CrisisHint` | **关键分** + 对手某招已用 ≥2 次 |

问题：OppCoachHint 和 LowEnergyHint 同时触发（非关键分 + 体力低 + 对手有惯用招）时，界面出现两条平行提示，占用约 40px 空间，且信息优先级不清晰。

**实现：四路分支 `CombinedHint`**

```jsx
function CombinedHint({ matchStats, score, phase, playerMoves, pEnergy }) {
  if (phase !== 'cards' || isKeyPoint(score)) return null;
  // ... 读招教练条件 + 低体力条件 ...

  // 两个都触发 → 合并
  if (hasOppHint && hasLowEnergy) {
    const sliceCounters = counter === 'slice';  // 一举两得特判
    return <div className={`bt-combined-hint ${tier}`}>...</div>;
  }
  // 只有读招教练 → 原样返回 bt-coach-hint
  if (hasOppHint) { return <div className="bt-coach-hint">...</div>; }
  // 只有低体力 → 原样返回 bt-low-energy-hint
  if (hasLowEnergy) { return <div className={`bt-low-energy-hint ${tier}`}>...</div>; }
  return null;
}
```

**「一举两得」特判**（`counter === 'slice'`）

当对手惯用招的克制招正好是「切削球」时（切削球本身是唯一回体招），合并文案升级：

> 体力告急（18）！「切削球」-3 续命，**且**克制对手惯用「上旋球」(×3)——一举两得

这比分开两条更有策略价值：玩家立刻明白切削球在此刻是双重最优解。

**视觉规格**（`bt-combined-hint`）

- `warn` 层级（体力 15-30）：橙色边框 `rgba(251,146,60,.35)`，背景 10% 透明橙
- `critical` 层级（体力 <15）：红色边框 `rgba(239,68,68,.40)`，背景 12% 透明红 + `tennis-bounce` 动效
- `reduced-motion`：压制 critical 动效（已加入 @media 规则）

**渲染层变化**

```jsx
// 之前（两行）
<OppCoachHint matchStats={...} score={...} phase={...} playerMoves={playerMoves} />
<LowEnergyHint pEnergy={state.pEnergy} phase={...} playerMoves={playerMoves} />

// 之后（一行）
<CombinedHint matchStats={...} score={...} phase={...} playerMoves={playerMoves} pEnergy={state.pEnergy} />
```
