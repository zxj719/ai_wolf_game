## 二、本轮改动

### 核心功能：角色选择屏新增打法标签

**问题定位：**

新玩家在 SelectScreen 面对 7 个角色时，仅能看到 emoji + 名字，对「选谁更适合我的打法」毫无判断依据。之前的 PrepScreen tip 优化（轮 19）帮助了已经上场的玩家，但选角阶段的信息盲区仍存在。

**数据复用：**

`moves.js` 的 `CHAR_BUILDS` 对象已经有 `style` 字段（重炮流、控制流等），是角色打法的权威定义，此前只被 `BattleScreen` 内部消费用于显示。本次直接 import 到 `SelectScreen` 复用，无需在 `gameData.js` 重复定义。

**实现方案：**

1. **下拉菜单**：option 文案从 `🦊 Elza` 改为 `🦊 Elza · 控制流`，在点击前就能看到风格
2. **角色卡片**：在 `.nm` 下方新增 `.chip-style` 小标签，optic 黄色等宽字体显示打法流派
3. **CSS 设计**：使用 `var(--mono)` + `var(--optic)` 0.75 opacity，视觉层次：名字（白/展示字体）→ 打法（optic/等宽），类似卡牌游戏的角色属性徽标

**打法标签一览：**

| 角色 | 标签 | 核心配招 |
|------|------|----------|
| 🐯 诚 | 重炮流 | flatDrive / smash / topspin / passingShot |
| 🦊 Elza | 控制流 | slice / dropShot / lob / volley |
| 🐰 菲比 | 灵动流 | volley / dropShot / passingShot / topspin |
| 🦖 Ross | 暴力流 | flatDrive / smash / passingShot / slice |
| 🍳 铁蛋 | 铁壁流 | slice / lob / topspin / flatDrive |
| 🐱 丫 | 网前流 | volley / smash / dropShot / lob |
| 🦢 莹 | 优雅流 | topspin / slice / lob / passingShot |

**改动文件：**

| 文件 | 操作 | 说明 |
|------|------|------|
| `src/modules/tennis/components/SelectScreen.jsx` | 修改 | import CHAR_BUILDS；option 文案加 style；chip 新增 .chip-style span |
| `src/modules/tennis/tennis.css` | 修改 | 新增 `.chip-style` 样式（6 行 CSS） |
