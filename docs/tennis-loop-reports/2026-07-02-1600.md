## 三、本轮改动

### 功能：FirstMatchHint「均势对阵」变体（bt-fmh-balanced）

**背景与动机**

R116 加入了 `FirstMatchHint`（BattleScreen 首局第 1 球提示），用于在玩家第一次对战某对手时，给出克制招建议。R118-R119 建立了均势信息系统：
- R118：Select 和 Prep 两处，均势→绿色提示（⚡）
- R119：OppCoachHint 和 CrisisHint，均势→绿色提示（⚡）

唯一遗漏的是 `FirstMatchHint`：当 `!counter` 时直接 `return null`。在莹 vs Ross 等均势对阵中，玩家在首局第 1 球位置看不到任何提示（SelectScreen 已有 ⚡ 绿标，PrepScreen 已有 ⚡ 绿框，但 BattleScreen 第 1 球静默）。

**完整 matchup 信息链（修复前后对比）**

```
阶段                  | 克制对阵          | 均势对阵（修复前） | 均势对阵（修复后）
---------------------|------------------|--------------------|-------------------
SelectScreen          | 🎯 克制招（琥珀）  | ⚡ 均势（绿）R118   | ⚡ 均势（绿）✅
PrepScreen            | ⚔️ 克制参考（琥珀）| ⚡ 均势建议（绿）R118| ⚡ 均势建议（绿）✅
BattleScreen 第1球    | 🆕 初次对战（绿）  | ❌ null 静默        | ⚡ 均势对阵（绿）✅  ← 本轮
BattleScreen 持续     | 🎯 OppCoachHint  | ⚡ 均势（绿）R119   | ⚡ 均势（绿）✅
BattleScreen 关键分   | CrisisHint 克制  | 注意读招 R119       | 注意读招 ✅
ResultScreen 复盘     | prep-growth-hint  | ✅ R112 已有        | ✅
```

**设计细节**

均势变体复用 `.bt-first-match-hint` 容器（flex 行、padding、border-radius），通过 `.bt-fmh-balanced` 修饰类覆盖颜色：
- 容器：`rgba(134,239,172,.05)` 背景 + `rgba(134,239,172,.20)` 边框（比克制变体的 `rgba(74,222,128,...)` 更低饱和度）
- 徽章 `⚡ 均势对阵`：`rgba(134,239,172,.78)` 文字 + `rgba(134,239,172,.10)` 背景
- 正文：`rgba(134,239,172,.72)` 颜色

颜色体系维持 R117-R119 规范：
- 琥珀黄（`rgba(251,191,36,...)`）= 有直接克制，可执行的具体行动
- 绿色（`rgba(134,239,172,...)`）= 均势/无直接克制，提示宏观策略

## 四、实现路径

### `src/modules/tennis/battle/BattleScreen.jsx`

`FirstMatchHint`（R116 原版 21 行 → 现 33 行）：

将原 `if (!counter) return null;` 替换为均势分支：

```jsx
if (!counter) {
  return (
    <div className="bt-first-match-hint bt-fmh-balanced">
      <span className="bt-fmh-badge">⚡ 均势对阵</span>
      <span className="bt-fmh-text">
        {opponent.face}{opponent.name} 与你无直接克制——读招节省体力，三维均衡加点是关键
      </span>
      <button type="button" className="bt-fmh-close" onClick={() => setDismissed(true)} aria-label="关闭">×</button>
    </div>
  );
}
```

原有克制分支（`🆕 初次对战`）保持不变。

### `src/modules/tennis/tennis.css`

在 `.bt-fmh-close:hover` 之后新增 3 条规则：

```css
.tennis-scope .bt-first-match-hint.bt-fmh-balanced { background: rgba(134,239,172,.05); border-color: rgba(134,239,172,.20); }
.tennis-scope .bt-first-match-hint.bt-fmh-balanced .bt-fmh-badge { color: rgba(134,239,172,.78); background: rgba(134,239,172,.10); }
.tennis-scope .bt-first-match-hint.bt-fmh-balanced .bt-fmh-text { color: rgba(134,239,172,.72); }
```
