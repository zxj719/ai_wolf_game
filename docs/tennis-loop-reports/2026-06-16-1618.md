## 二、本轮改动

### 核心功能：OppIntelCard 对手属性预览

**问题定位：**

PrepScreen 的对手情报卡（`OppIntelCard`）仅展示对手的打法风格标签和四招招式名，但不显示对手的实际属性数值（sta/skill/mind）。这造成两个问题：

1. 玩家无法判断对手是「力量型（高 sta）」还是「技巧型（高 skill）」——而这对备战加点方向有直接影响
2. 当对手的惯用招式与其高属性结合时（如心态 88 + lob），玩家没有意识到威胁有多大

实际上 `state.opp.sta/skill/mind` 已在 PrepScreen 的 props 中，仅需渲染。

**实现方案：**

在 `OppIntelCard` 底部新增 `.opp-stats-strip` 区块，显示对手三维属性条：

1. **数据来源**：`opp.sta`、`opp.skill`、`opp.mind`（40–90 范围），条形宽度以 `val%` 直接映射（100 = full bar）
2. **视觉设计**：
   - 橙色渐变（`var(--clay)` → `#ffb27a`）刻意区别于玩家属性的绿色渐变——让「己方属性」和「对手属性」在视觉上天然分离，无需额外标签
   - 标签行：emoji 图标 + 文字标签（52px 固宽）+ 属性条（flex:1）+ 数值（右对齐 monospace）
   - 属性条高度 8px，比玩家面板的 12px 略细，视觉层次上低于玩家面板优先级
3. **提示文案更新**：`了解对手惯用四招，针对性强化弱项属性` → `了解对手惯用四招和属性分布，针对性强化弱项`

**策略意义举例（帮助玩家理解）：**
- 对手 `mind: 85 + lob（心态系）` → 提示玩家加 sta 或 skill 来抗衡
- 对手 `skill: 45`（低技巧）→ 对手的 topspin/dropShot 不强，可以放心面对
- 对手三维均衡 → 提示无明显弱点，需要全面布局
