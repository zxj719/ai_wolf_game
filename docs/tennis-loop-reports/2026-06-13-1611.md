## 二、本轮改动

### ✅ TennisRoute.jsx — 成就图鉴锁定格

**修改前（仅显示已解锁）：**
```jsx
{progress.achievements.length > 0 && (
  <div className="rule-strip" style={{ marginTop: 0, marginBottom: 14 }}>
    {ACHIEVEMENTS.filter((a) => progress.achievements.includes(a.id)).map((a) => (
      <span key={a.id} title={a.desc}>{a.icon} {a.name}</span>
    ))}
  </div>
)}
```

**修改后（全量展示，锁定 vs 已解锁）：**
```jsx
<p className="hint">
  💰 {coins} · 👑 球王 ×{champs} · 📖 绝技图鉴 X/7 · 🏅 成就 {progress.achievements.length}/{ACHIEVEMENTS.length}
</p>
<div className="ach-gallery">
  {ACHIEVEMENTS.map((a) => {
    const unlocked = progress.achievements.includes(a.id);
    return (
      <span
        key={a.id}
        className={`ach-chip${unlocked ? ' ach-chip--on' : ''}`}
        title={unlocked ? a.desc : `🔒 ${a.desc}`}
      >
        {unlocked ? a.icon : '🔒'} {a.name}
      </span>
    );
  })}
</div>
```

**设计选择：**
- 锁定格 tooltip 加 `🔒` 前缀提示未解锁状态，同时显示解锁条件 → 玩家知道怎么解锁
- `opacity: .55` 锁定态保持可读性（.3 太暗看不清成就名）
- hover 到 `.75` 提示可互动（有 tooltip 信息）
- 已解锁用 `rgba(245,166,35,.18)` 琥珀半透明背景，与暗色主题协调
